Job Templates Overview

Purpose:

Job Templates enable you to define various types of standard job, site and equipment structures that facilitate the creation of new customers and/or Sites from the Job Record or from New Call.

Once the Templates have been created in the special Templates viewers (for Clients, Sites and Jobs), they are used in regular Job (and New Call) viewers for Cloning. A Template may be selected in the Job viewer for Cloning – but the selected Template cannot be changed in the Job viewer.

Jobs should NOT be cloned from the Job Template viewer.

Accessing the Templates Viewers:

In order to access the Job, Client Site and Client Template viewers, your user group must have rights to these viewers. It is recommended this ability is limited to supervisory or other advanced users.

Setting Up Your Templates for Jobs that will NOT use New Call 2/COD Processing:

Step 1: Decide on your Job Structures

Decide how your various template jobs should be structured in terms of cost codes, billing items and so on. You may wish to have several types of templates – for residential clients, commercial clients, and so on.

Step 2: Create the Client Template Record

Create a Client Template record in View Client Templates. The Template box must be checked. Only those fields that will be common to ALL clients generated from this Template should be filled in. All other fields in this template record (except those required to save the record) should be blank.

Step 3: Create the Client Site Template Record

Create a Client Site Template record in View Client Site Templates. The Template box must be checked. Only those fields that will be common to ALL sites generated from this Template should be filled in. All other fields in this template record (except those required to save the record) should be blank.

Step 4: Create the Job Template Record

Create a Job Template record in View Job Templates. The Template box must be checked. Set the Client and Client Site Id’s to the Client and Client Site Templates you just created.

Aside from the Client and Client Site fields, only those fields that will be common to ALL jobs generated from this Template should be filled in. All other fields in this template record (except those required to save the record) should be blank.

Step 5: Create the Job Billing Item Record(s)

Create at least one Job Billing Item record for each of your template jobs. The information on this record will be used for invoicing. All Billing Items associated with the Job will be cloned during the New Call Create and Clone Job Wizard processes. Typically, the Invoice and Billing Worksheeting fields would be completed.

Step 6: Create the Job Cost Code Record(s)

Create at least one Job Cost Code Billing Item record for each of your your template jobs. All Cost Codes associated with the Job will be cloned during the New Call Create and Clone Job Wizard processes.

Step 7: Create a Site Equipment Record

Create at least one Site Equipment record for each of your template sites. Be sure to complete the Job fields in this record with your Template Job.

Using your Template Files with Job Clone Wizard

You may select a Template Job in the regular Job Account Viewer. You will not be able to edit the Template, but you will be launch the Clone Job wizard. When the Job has been cloned, it will appear in Job Viewer so the you can easily edit it and its associated record.

Using your Template Files With New Call Create (not New Call 2 / COD)

Once your template records have been created, using them in New Call is as simple as A-B-C.

Step A: Verify the Customer is Not Already in your Database

Use the New Call's Search Text feature to verify that the customer is not already included on your database: be sure to search by Name and at least one other parameter, such as phone number or street address.

Step B: Select the residential Client Site and Switch to New Call Create

Once you are sure that the customer is not in your database, select the residential site on the Client Site list in New Call.

With residential selected, click the Search button to turn Search Off and Create on.

Step C: Enter the New Client Site, Equipment and Work Order Information

In the New Call Create screen, type over the Client Site Id residential with your firm's id for this client site. Enter the work order and equipment fields as discussed in the New Call Create Help Topic. The Client Id and Name must remain blank: when you click the Add record button, the system will also create a new client record with the same Id as that of the new client site.

Using Template Files with New Call 2 / COD Processing

Create the Client Template Record

Create a COD Client Template record in View Client Templates. In COD processing, AR will be applied against this record: individual Client Records will not be created for each customer.

The Template box must be checked.

In the Client Address, enter COD / Do Not Mail.

Create the Client Site Template Record

Create your COD Client Site Template records in View Client Site Templates.

The Template must reference an AR Terms Id that has the COD box checked; these are the only Template Sites that will appear on the New COD Site screen.

The Template box must be checked.

Only those fields that will be common to ALL sites generated from this Template (In New Call 2 / COD processing) should be filled in. All other fields in this template record (except those required to save the record) should be blank.

To accommodate taxation issues, you may need to create multiple Template COD Sites – one for each sale tax region (tax group) you will serve.

Job Templates

Create your COD Job Template records in View Job Templates.

When creating Job Templates for COD Sites, the Site Id in the Job record must be blank. New Call 2/COD is smart enough to use the Job referenced in the Equipment record you select in the New Call process.

To accommodate taxation issues, you may need to create multiple Template COD Jobs – one for each sale tax region (tax groups) you will serve.

The Template box must be checked.

On the Invoice Tax, copy the Client address.

Job Billing Item Record(s)

Create at least one Job Billing Item record for each of your template jobs. The information on this record will be used for COD invoicing. Typically, the Invoice Report Query fields would be completed.

Job Cost Code Record(s)

Create at least one Job Cost Code record for each of your COD template jobs.

Create a Site Equipment Record

Create at least one Site Equipment record for each of your template COD sites. Be sure to complete the Job fields in this record with your Template COD Job.
