Registry Tab Mismatch Alert

This recommended registry entry below should be implemented to write tab mismatch information to the error log.

Setting Values are Uppercase-Lowercase Sensitive:
The Setting Value is case-sensitive: be sure that you enter the value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.

Note that, if implemented Audit Changes information (from secure notes that are not saved because the record is not saved) are now written to the error log. Information on accepted and rejected charges are recorded: rejected entries will be flagged with *REJECTED*.

These registry entries should be implemented to record tab mismatch errors to the error log.

Id: sys-alert-tabmismatchhistory

Setting Key: errorlog

Setting Key: messagebox

Note that the above options do not require Setting Values.
