Registry W2 Electronic Files / Magnetic Media (sys-w2-ef)

Purpose:

This registry entry is required if you will be filing your Federal W2’s electronically (as magnetic media) via the Report W2 Electronic Files. Registry options (listed below) may be used for the state of Oklahoma and the city of Philadelphia.

This registry is also used by the Report W2 Electronic Files (Maryland) - see the Additional Settings for Maryland State Filing section of this document.

Important Information for Intercompany Users:
Report W2 Electronic File (RW2EFv82b01.rpt dated 2020-01-08 or later) has been updated to use the Company record / Company Registry tab to specify the Federal EIN, Company Name and Address and other registry settings required for these reports. The Registry Entries made on the Company's Registry tab will override those made in the Registry Viewer.

Test Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Setting Values are Uppercase-Lowercase Sensitive
The Setting Value is case-sensitive: enter the Value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.

Usage:

A related registry entry, Registry Entry W2 Electronic Filing Report, must also be completed for state filing (including Maryland) and federal filing. If you are using the report RW2EFv82b01.rpt dated 2020-01-08 or later these entries may be entered in the Company Registry tab for Intercompany users.  

The Registry Entry Federal and State EIN and Registry Company Name Entry must be also be set correctly.

The most recent copy of RW2EF.rpt must be imported into your database.

If you will be submitting W2's to the State of Maryland, the report file RW2EF-MD.rpt must also be installed.

If you will be submitting W2's to the State of Pennsylvania the report file RW2EF-PA must also be installed.

See Report W2 Electronic File for instructions on running this report.  See Report W2 Electronic File (Maryland) for details on the Maryland report.

Additional Settings For Maryland State Filing

The following additional settings are required by the Maryland state report. A special version of this report, Report W2 Electronic File Maryland (RW2EF-MD.rpt) should be used for Maryland.  

 Id: sys-w2-ef    (Note that the standard registry Id is used even though the Report Id is different)

Setting Key: reMDCentralRegistrationNumber  Setting Value: obtain this code from Maryland

Eight-digit Maryland Central Registration Number (Tax Withholding Account Number).

Setting Key: rvRepresentativeName   Setting Value first and last name

The individual authorized to certify the filing of this report.

Setting Key: rvRepresentativeTitle   Setting Value title

Enter the title of individual authorized to certify this report.

Setting Key: rvRepresentativePhone  Setting Value: required

Enter the phone number of the individual authorized to certify this report (1234567890).

Setting Key: rvNAICSCode  Setting Value: a valid NAICS Code

The North American Industry Classification System (Business Activity Code) for your company. Enter a 6 digit code.
