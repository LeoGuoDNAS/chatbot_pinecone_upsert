Labor Journal Time Log Tab

This tab is used in conjunction with TimeAnywhere to record employee in and out times. It is populated when time is posted from TimeAnywhere.

Start Date:
The day being logged. Each day runs from 12:00 am (midnight) to 11:59 pm.

Start Time: End Time:
The Starting and Ending times in 24-hour (military) format. Note that the End Time must exceed the Start time. Note that if time is being recorded for a shift that spans midnight, two lines must be created: one for the hours before midnight, and another on the following day for the hours following midnight.

Description:
A brief description of the work being done.
