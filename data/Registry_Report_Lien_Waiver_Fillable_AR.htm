Registry Report Lien Waiver Fillable AR

Purpose:

SAMPro's Fillable Form functionality allows an AR Lien Waiver to generated as a fillable PDF.  This uses the boilerplate information included in a standard AR Lien Waiver to be combined with the information from SAMPro to quickly and easily generate the document. This Lien Waiver may then be emailed to the customer when payment is received.

Important: Note that this feature requires use of the XmlToPdf[2019-01-16 Java 8-11].jar file (or higher).  

The following tasks are needed to support the Fillable Form:

1. The Report Lien Waiver Fillable (AR) must be installed.

2. The following Registry must be implemented:

Id: sys-report-RLNWVRFLLBLEAR

Setting Key: autosavefiletype   Setting Value: PDF

Setting Key: autosavefilename   Setting Value: =^sectionkeyvalue^,^date^,^time^

The ^sectionkeyvalue^ token consists of the Job Id-Invoice Id followed by the type of waiver specified when the report is run (conditional or final). In this example, the sectionkey is followed by a date and time stamp; if the date and time tokens are not included, a registry to allow autosaveoverwrite should be added.

Setting Key: autosavebysection Setting Value: true

This option is used to save the report sections as individual files.

Setting Key: autosavedirectory Setting Value: path   

Where path is replaced by full path to the existing directory that you want your reports stored in.

Setting Key: fillableform   Setting Value: true

Required for fillable forms.

Setting Key: sectionfillableform  Setting Value: true

Required for fillable forms to be generated by section.

Setting Key: sectionfillableformflatten  Setting Value: Y

This optional setting allows the PDF to be 'flattened' so that it may not be edited after creation.

Setting Key: FillableFormPathConditionalProgress

Setting Value: path\ConditionalWaiverAndReleaseOnProgressPayment.pdf

This entry provides the full path to the pdf template used for the conditional waiver and release.

Setting Key: FillableFormPathConditionalFinal  

Setting Value: path\ConditionalWaiverAndReleaseOnFinalPayment.pdf

This entry provides the full path to the pdf template used for the final waiver and release.

Option to Prevent Auto-Open of PDF

Setting Key: showdocumentfillableform   Setting Value: false (default true)

If this registry is not in place, clicking the Fillable Form button to generate the formatted, PDF version of the report will open the PDF. If the above registry option is in place, the PDF will be created but not opened.

3. The template PDF files must be located in the directory specified by the Setting Values above. Sample templates may be obtained from Data-Basics.  The text and signature graphic of the Claimant may be adjusted as needed.

Important: Be sure that the sample template is approved for your state; some states require a notary. The individual/attorney responsible for Lien Waivers at your firm should carefully review the proposed template before it is implemented.
