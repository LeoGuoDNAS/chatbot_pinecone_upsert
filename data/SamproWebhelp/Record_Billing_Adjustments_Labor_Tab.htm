Record Billing Adjustments Labor Tab

Purpose:

The Labor tab enables you to verify that the labor hours on this invoice (that were entered in the Labor Journal, Remote Labor Journal, or Work Order Labor Tab) are recorded correctly, and to make any adjustments required. Be sure that the sales tax type is set correctly. Items flagged as Hold will not be included on the invoice.

Note that this tab is NOT displayed for GMP Contract and SoV Jobs.

Usage:

Note that labor lines can only be adjusted, not added.

Important Fields:

Click the Magnify button to display all fields for this line.

Next Inv. Hours
This field defaults from the hours entered via the Labor Journal, Remote Entry, or the Work Order Labor tab. It may be adjusted by the user if needed.

Revenue Rate
The rate at which these hours are being charged. It may be adjusted by the user if needed.

Unit Price:
This display-only field is maintained by the system.

Next Inv. Labor
 The Labor dollars to be included on the next invoice (Next Invoice Hours x Revenue Rate).

Labor Rate Id:
The labor rate for these hours. It may be changed by the user if needed.

Bill Type Id:
The Bill Type Id for hours are being charged. It may be adjusted by the user if needed.

Description
This field will appear on this invoice if your invoicing format is set up to include it.

Tax Type Id
This field should default from the job and sales tax authority records but may be adjusted by the user if needed.

Hold
If this box is checked (by double-clicking or pressing the spacebar when the caret is on it), this labor line will not be included on the next invoice (for example, if you were waiting for a certain task to be completed before billing these hours, or if these hours were in dispute).

Extra
If these hours are "extras" on a fixed fee contract billing, this box should be checked.
