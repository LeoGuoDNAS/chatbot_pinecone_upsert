Company Viewer

Purpose:

This record is required for Intercompany Processing. The Company record defines the Security Id, GL Entity (for balance sheet transactions), and ‘Due To’ and ‘Due’ From GL Accounts.

See Intercompany Processing Overview for additional implementation details.

Registry Entries:

See Registry Intercompany Processing for the global registry option required to implement Intercompany Processing.

Registries for certain US Government reports may be entered on the Company's Registry tab. See the Registry Tab for details.

Related Registry:

See Block Users from Seeing Records with Specific Security for a registry to limit users' access to specific companies.

Important Fields: Upper Section

Main Tab

Company Id:
Uniquely identifies this company. It is recommended that the Company Id and Security Id match.

Company Name:
 The Name of the Company. If a report is run for a selected company, the company name on the report will default from this field. If a report is run for all companies, the company name on the report will default from the companyname registry entry.

Security Id:
This required field specifies the Security applied to transactions for this company.

Federal EIN:
The Federal EIN for this Company.

The Federal EIN and Company Name are fields are used by the following reports to set the Company EIN and Company Name, superceding the company-specific registry settings:

Report AP 1099's (RAP1099v82b01 dated 2020-01-17 or later)

Report AP 1099 Electronic File (RAP1099EFv82b01.rpt dated 2020-01-17 or later)

Report W2 Electronic File (RW2EFv82b01.rpt dated 2020-01-17 or later)

Report W2 / W3 (Standard and old)

Report StateUnemployment Electronic File (RSTTEUEv82b01.rpt dated 2020-01-17 or later)

GL Tab

Balance Sheet Default: GL Entity Id:
The GL Entity to be applied only to Intercompany transactions. This GL Entity should NOT represent a profit center.

Intercompany DUE FROM this Company: GL Account Id:
The Asset account to be debited in Intercompany transactions.

Intercompany DUE TO this Company: GL Account Id:
The Liability account to be credited in Intercompany transactions.

Address Tab

Mailing Address:
Street Address:
City: State: Zip:

The address of this Company. These fields are used by the following reports to set the Company Address fields, superceding the company-specific registry settings:

Report AP 1099's (RAP1099v82b01 dated 2020-01-17 or later)

Report AP 1099 Electronic File (RAP1099EF dated 2020-01-17 or later)

Report W2 Electronic File (RW2EFv72b105 dated 2020-01-17 or later)

Report W2 / W3 (Standard and old)

Report StateUnemployment Electronic File (RSTTEUE dated 2020-01-17 or later)

User Fields Tab

Company User 1-5:
These optional fields enable you to retain additional information about this Company.

The Key and Value fields may be used to set the Setting Key and Setting Value registry options for certain government report as listed below  We strongly recommend specifying the report the registry option supports in the Description field.

States Tab

ST:

This tab should be used to define the State/States associated with this Company.

State EIN:

This field is reserved for future use with reporting.

Entities Tab:

This tab displays the Balance Sheet and Profit Center Entities that point at (belong to) this Company record.
