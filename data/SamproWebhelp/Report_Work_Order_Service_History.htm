Report Work Order Service History

Purpose:

This system report provides information on the services performed, parts used and metrics taken for one or more Site.

Registry:

Since the Work Order Age and Max Calls fields are required, you may wish to provide initial values for them via Registry.

Important Fields:

Sort By: Work Order Id - Equipment Id
Select Work Order Id to sort the information for each Site by Work Order. Â Select Equipment Id to sort by Equipment, then Work Order.

Work Order Age (Days):
This required field specifies the maximum age of the work order in days.

Max Calls:
This required field specifies the maximum number of Work Orders on the report.

Show Parts - Show Misc. - Show Metrics:
Select one or more option to include this information on the report.

Changed Since:
This optional field limits the report to Work Orders changed since this date.

Show For WO Recommendation:
Show From WO Recommendation:
Show Unacknowledged Recommendation:
Select One or more of these options to include Weblog information on the report.
