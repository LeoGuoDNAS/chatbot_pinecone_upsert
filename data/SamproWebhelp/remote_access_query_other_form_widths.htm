Remote Access Query: other_form_widths

Purpose:

Controls the widths of the cells in the Misc. Form on the Add/Edit Misc page.

Usage:

By default, the elements in the Misc (Other) Form are set to the following widths in pixels: [100,100,200,80]. This RAQ enables you to change these specifications or to allow the system to automatically accommodate the character widths.

Important Fields:

Application: TechAnywhere

Dataset Name: other_form_widths

Query Id: This field is assigned by the system when the record is saved.

Query Format: Frame

DataSet Query:

To automatically accommodate the character widths of the table (open bracket close bracket, no space in-between):

 []

To specify the width of the Quantity, Part Number, Lookup, Equipment Id and Location Id columns:

 [pixels1,pixels2,pixels3,pixels4]

 where pixels1-4 are the replaced by the width in pixels.

  Example: [110,110,250,80]
