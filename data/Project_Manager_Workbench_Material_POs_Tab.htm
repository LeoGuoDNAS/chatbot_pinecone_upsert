Project Manager Workbench Material POs Tab

Purpose:

This tab lists the Inventory and General information from Purchase Orders associated with this Job.

Usage:

See Project Manager Workbench for details on this viewer’s upper tabs.

Important Fields:

PO Id – Description – QtyOrdered – U-M Unit Cost – Amount – QtyReceived – QtyVouchered – Vendor Name

This tab is updated as PO’s are entered, cancelled, received and vouchered.
