Registry Flat Rate

Flat Rate Import from Profit Strategies and Cool Front

The following registry setting allows a fourth level of flat rate pricing imports to be enabled.  Data will be imported from Third-party developer Profit Strategies or Cool Front files using the Start Kimmons1 File Import function.

Id: sys-viewer-vfltrte

Setting Key: UseKimmonsFileLayout2   Setting Value: true   (default is false)
