Report COP Quote

Purpose:

This report prints COP Orders flagged as Quote or ProForma.

Usage:

COP Orders flagged as ProForma will be printed using the ProForma Table (quantity and currency conversion table) specified in the Client Site/Client Order record. This produces a document designed for international shipping.

Important Fields:

Report Options:
The Report Options selector pane controls how the report will be output. Click Report Options to learn more.

Report Ranges:
The Report Ranges selector pane enables you to select a customized range for the report. When you click on a range, the criteria defining it will appear in the ID field(s). Creating a customized report range is discussed below. Notice that you must name a custom range in order to save it.

Range:
Enter a unique, alphanumeric name to save and identify your customized report Range. Ranges are defined by the contents of the ID field(s).

Order Id:
Enter the Id of the Quote or ProForma order you wish to print (or double-click to select from a list). Several Orders can be specified by using a comma to separate the individual ID's. A range of Orders is defined by the first and last Id's of the range separated by an underscore.

Client Id:
To print orders flagged as Quote or ProForma for a specific Client, enter the Client Id (or double-click to select from a list). Several Clients can be specified by using a comma to separate the individual ID's. A range of Clients is defined by the first and last Id's of the range separated by an underscore.

Date Ordered:
Enter the Date Order (in yyyy-mm-dd format) of the Quote or ProForma Order you wish to print (or double-click to select from the Calendar viewer). A range of Dates is defined by the first and last Date of the range separated by an underscore.

Entered By:
Enter the User Id of the person that created the Quote or ProForma order.

Lines:
This informational field displays the number of lines per page for this report under the current font setting.

Multiple Fields
You can optionally enter Id's in more than one field. Items will appear on the report only if they fit all the criteria set in the Id fields.

Power-User Reporting Tip:
You may also use the comparison operators >,<, = and ! in your report ranges to control the information output to your report. Click Comparison Operators for details.
