Registry Start Document Archive Maintenance

Purpose:

This registry entry provides the default values for the manual Start Document Archive Maintenance function and the Start Document Archive Maintenance (Automated) function so that it may be run as a Scheduled Event.

Related Registry:

See the Subfolder Option for Document Archiving in Registry Document Archiving (sys-document-archive) for enabling Subfolders when documents are first archived.

Important Fields:

Id: sys-start-sdcmntarchvemntnnce

Specify Default Days Past to Check PDF Archives

Setting Key: NumberOfPastDaysToCheckPDFArchives   Setting Value: numberofdays (default is 7)

This registry entry provides the default value for this field for the manual Start Document Archive Maintenance function and the Start Document Archive Maintenance (Automated) function.

Maximum Documents Options

Setting Key: MaxDocumentsToProcess Setting Value: numbertoprocess

A reasonable value to start with is 1000. If your system handles this well, the number may be increased. This field is required if the Document Archive (Automated) will be run from Scheduled Event Manager

Setting Key: RequiredFieldsParent  Setting Value: dcmnt-mx

This requires a value in the Maximum Number of Documents to Process field.

Move to Subfolders

Setting Key: MoveToSubfolders Setting Value: true (default false)

This recommended option moves files to subfolders by the Month/Year the document was attached.

Convert Reports to PDF's

Setting Key: ConvertDBRsToPDFs Setting Value: true (default false)

This setting converts archived documents from the native SAMPro DBR format to PDF's for easier distribution.

Log Settings

Setting Key: ShowDetails Setting Value: true

This option controls the level of detail written to the log file generated when this function is run. It should always be enabled.

Force Archive

Setting Key: ForceArchive  Setting Value: true

This registry sets the default value of the Force Archive option to checked for the manual Start Document Archive Maintenance function and the Start Document Archive Maintenance (Automated) function. Typically, this would only be set if there was a problem with the loss or corruption of archived files. Shouldn't be used for normal archive maintenance cycles.
