Purpose:

This record is referenced in the Job Cost Category. It is currently used in conjunction with the Avalara Web Tax service.

Important Fields:

Cost Category Group Id:
This required field uniquely identifies this record.

Note: Avalara Web Users
A Job Cost Category Group should be created with the Id of Avalara.   This record should be reference by Service Invoice Job Cost Categories. 

For COP, the Inventory Groups must include a Job Cost Category with an Avalara code of NT if items in the group are to be non-taxed.  By default, if no Job Cost Category exists in the Inventory Group, all items will be taxed.
