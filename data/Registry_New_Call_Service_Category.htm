Registry New Call Service Category

Purpose:

Warn the user if more than n WorkOrders have been created in y days for the specified service category.

Usage:

You must create one registry record for each Service Category that should generate the warning.

Related Registry Entries:

Registry Entry for New Call Options

Registry to define Disabled / Required Fields in New Call

Important Fields:

Id: sys-newcall-ServiceCategoryId

For example, sys-newcall-HVAC would be used if you wished to issue the warning for Service Category HVAC.

Important: Both of the following Keys are required or the registry setting is ignored.

 

Setting Key: WorkOrders   Setting Value: #

Setting Key: Days   Setting Value: #

 

For example:

Id: sys-newcall-HVAC

Setting Key: WorkOrders   Setting Value: 3

Setting Key: Days   Setting Value: 60
