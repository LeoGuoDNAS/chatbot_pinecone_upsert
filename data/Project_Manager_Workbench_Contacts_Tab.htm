Project Manager Workbench Contacts Tab

Purpose:

The Contacts tab enables you to attach one or more Contacts to the selected Job. Changes made to this tab will be reflected on the Job viewer.

Usage:

Contacts are used for generating RFI, Transmittal and Submittal documents. See Project Manager Workbench for details on this functionality.

See Contact Viewer for details on the Contact record.

Important Fields:

Contact Id:
Enter a valid Contact Id (or double-click to select from a list). You may create as many lines as needed to accommodate all contacts. The display-only Name, Contact Group, Email, and Phone number fields will default from the Contact record.

Contact Type Id:
This optional field defaults from the Contact record but may be changed by the user if needed.
