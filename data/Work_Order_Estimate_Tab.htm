Work Order Estimate Tab

Purpose:

This tab may be used to record estimates for future service, installation and other projects. A running total of the cost and revenue is displayed on the top of the tab.

Usage:

This tab is used for reporting purposes only, and is deprecated in favor of Quotemaster.

Important Fields:

Quantity
The number of this item/charge being estimated.

Other Rate Id:
Enter a valid Other Rate Id defining this estimate or double-click on this field to select from a list. The Description field defaults from the Other Rate record, but may be modified by the user if desired.

Unit Price
The Unit Price of this estimate.

Extended Price:
The Unit Price X Quantity. The system will calculate this field for you.

Unit Cost:
The Unit Cost of this estimate.

Extended Cost:
The Unit Cost X Quantity. The system will calculate this field for you.

Description:
This field defaults from the Other Rate record, but may be modified by the user to briefly describe this estimate.
