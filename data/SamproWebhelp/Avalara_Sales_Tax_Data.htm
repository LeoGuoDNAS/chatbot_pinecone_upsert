Avalara Sales Tax Data

This viewer displays the Sales Tax Data imported via the Start Sales Tax Update (Avalara) function.  

 

In SAMPro, if an inventory item is NOT flagged taxable, it should NOT calculate tax.

When the integration with Avalara is enabled, the calculation is no longer done by SAMPro. This means that whether or not an item is taxed is determined exclusively by Avalara, as it does not consider whether or not this is selected in SAMPro.

This flag will remain available to allow proper record keeping (for historical purposes as well as any other required reason).
