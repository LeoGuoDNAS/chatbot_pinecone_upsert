GL Budget Journal Viewer (Budget Entries)

Purpose:

Enter and maintain General Ledger Budget entries.

Also View:

Registry GL Journal (sys-viewer-vgljrnllst)

Usage:

Create budgets for Sales accounts (crediting by entity if desired). On the expense side, target G&A budgets can be set up, and compared with actual values on financial reporting.

Important Fields: Upper Section

Transaction Id:
This field will be completed by the system when you save this entry.

Total Debits:Â Total Credits:
These display-only fields are maintained by the system based on the detail that you enter in the lower part of the screen.

GL Account Id:
The GL Account to be affected by the budget entry. Enter a valid GL Account Id or double-click to select from a list.

GL Budget Transaction User 1: GL Budget Transaction User 2:
These optional fields may be used for informational or reporting purposes.

Important Fields: Lower Section

Entity:
The GL Entity affected by this entry. Enter a valid Id or double-click to select from a list.

Period Id:
The Accounting Period this entry is assigned to. Enter the Id of an active accounting period or double-click to select from a list.

Debit:
If the Entity on this line should be debited, enter the amount.

Credit:
If the Entity on this line should be credited, enter the amount.

Effective Date:
The Effective Date of the transaction (in mmddyyyy format); this date must fall within the range specified by the Accounting Period. Press Spacebar to enter today's date; the + and - keys may be used to increment or decrement the date.

Description:
Briefly describes this line. Although this field is optional, it is strongly recommended for informational and reporting purposes.
