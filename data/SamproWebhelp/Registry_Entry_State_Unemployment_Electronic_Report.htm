Registry State Unemployment Electronic Report (sys-ue-ef)

Purpose:

This registry entry is required if you will be filing your State Unemployment reports electronically. Kansas and Texas are currently supported.

Important Information for Intercompany Users:
The Report State Unemployment Electronic File (RSTTEUEv82b01.rpt dated 2020-01-08 or later) has been updated to use the Company record / Company Registry tab to specify the Federal EIN, Company Name and Address and other registry settings required for these reports.

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Setting Values are Uppercase-Lowercase Sensitive:
The Setting Value is case-sensitive: be sure that you enter the value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.

Usage:

In addition to the required registry settings (in the Registry Viewer or the Company Registry tab), your State EIN must be correctly entered in the Employer Id field of the Tax Authority record for all applicable states.

See Magnetic Media State Unemployment Reporting for details on running the report itself.

The following options are required for all states. The Setting Value must be completed for your company according to the specifications for the state or states you are filling with.

Note Phone and Fax Numbers: The contact phone and fax numbers should be entered as digits – do not include ( ) or a dash.

Wrong: Setting Key: raContactPhoneNumber  Setting Value: (555) 663-5600
