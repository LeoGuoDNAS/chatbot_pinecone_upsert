Start GL Report Export

Purpose:

Since GL reports are stored in your database, this new support function enables you to export one or many of your GL reports from your database to a directory on your disk. Reports should be exported to a folder for archival purposes before importing new ones. Your support representative will inform you if there are other times when it is necessary to export a report (typically, this would be done for support reasons).

Usage:

To export GL reports, complete the following steps.

1. To export your user-defined reports, create a directory you can export them to when you are running the Data-Basics software. For example, if you are running the software from c:, you might want to copy the report files to c:\glreports. If this directory contains earlier copies of reports, we suggest that you move these files to a backup location (c:).

2. Access the Start Export GL Report function. Fill in the Reports Directory as discussed below.

3. When you click the Go button, the system will display a checklist containing all your GL reports. You may select all or some of these reports for export.

4. When you click the Go button on the checklist, the system will begin the export. Once the export is complete, the reports will be saved to the directory that you specified.

Important Fields:

Reports Directory:
Enter the path to the directory containing the reports (for example, c:\glreports).
