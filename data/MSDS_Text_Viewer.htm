MSDS (Manufacturers Safety Data Sheet) Text Viewer

Purpose:

Create and maintain the cover sheet text that prints with your Manufacturers Safety Data Sheets. This viewer may also be used to create tag text that will print when a tag is encountered in the MSDS Master record for an item, located in the Inventory Item record's MSDS tag.

Usage:

The coversheet MSDS Text record is referenced during the printing of MSDS's. The coversheet Id is reserved for this purpose only, and should not be changed. Other MSDS Text records may be created and referenced as tags as needed: refer to the Inventory Item MSDS Tab for details.

If you are creating a new MSDS Text record, be sure to click the Add Record button before clicking the Note button. Notes can only be associated with an existing (saved) record.

If TSCA validation checking is enabled, MSDS sheets will be sent with every order if an item is not registered; click Registry Entry for TSCA Registration for details.

Important Fields:

Note: this Help Topic lists the standard configuration of all fields that may be included in this record. If you are using a customized system, the fields displayed on your screen may not exactly match the order or status (required, optional, default, display-only, hidden, etc.) of the fields described in this Topic.

MSDS Text Id: (15 character, only . and - special characters)
A unique Id. This field is required. Note that the Id coversheet is reserved for use as discussed above.

MSDS Text Description: (40 character)
A brief description; this field is optional.

By using Windows' clipboard facilities, you can easily copy and paste between the Notes for a particular record and your other Windows applications (such as word processing).

1. Verify that the MSDS Text record you wished to enter/edit text to has been saved (the Add Record button should be disabled).

2. Click the button entitled Note on the side button bar.

3. The Notes viewer for this record will automatically open.

4. If you have copied Windows text to the clipboard for this Note, choose the Edit menu (in the Notes viewer) and select Paste. The text that you copied to the clipboard will be inserted in the Notes viewer.

5. To save the notes for this record, simply close the Notes viewer (the system automatically saves the current contents for you).

6. Click the Change Record button on the MSDS Text viewer.
