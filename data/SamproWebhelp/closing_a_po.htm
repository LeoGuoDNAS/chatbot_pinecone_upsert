Closing a Purchase Order

Purpose:

A PO should be closed after it (and any required change orders) has been printed and all items on the order have been marked as received on the PO or in the Inventory Item Receipt Journal.

You can then close the PO by flagging the "Close PO?" box in the AP Journal.

In Version 7.1 and up, PO’s with a status of Released may now be marked as Closed in the Purchase Order viewer. This change conforms to the A/P functionality where a PO could be set to "Closed" when the PO status was “Released.”

Setting the Date Closed Manually

If your firm does not typically print purchase orders, the Date Closed may be set manually. Doing so will not change the Purchase Order’s status, but will prevent it from displaying in the active Purchase Order list.

PO Status Setting the Date Closed

Released You may enter the Date Closed. If the PO Period on the Main tab is inactive, the system will inform

that the period is inactive: click OK to confirm the message. The system will allow you to save the record with the Date Closed set.

Printed  When you enter the Date Closed and attempt to save the work order, the system will ask if wish to

make this a Change Order. Click No and save the PO; the Date Closed will be set.

If the PO Period on the Main tab is inactive, the system will also inform that the period is inactive: click OK to confirm the message. The system will allow you to save the record with the Date Closed set.

Changed If the PO has a Status of Changed, and the accounting period on the Main tab is active, you will be

able to set the Date Closed and Save the PO.

However, if the Status is Changed AND the Accounting Period is Inactive, you must do the following before setting the Date Closed:

1) Print the Purchase Order (this will set the status to Printed).

2) Click the Refresh button.

3) Enter the Date Closed and save the PO. You will be alerted about the inactive accounting period, but the system will complete the save with the Date Closed.

Pending Cancel If the status is Pending Cancel, you will not be able to set the Date Closed on the PO.

Print the Purchase Order; doing so will set the Status to Canceled.

Cancelled Cancelled PO’s will not appear on the list of list of Active PO’s, so the Date Closed need not be set.

However, should you wish to do so you may.

Closed  The Date Closed field is now automatically set when the status of the PO is set to Closed.
