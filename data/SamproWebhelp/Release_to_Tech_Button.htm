Release to Tech Button

 

When the Release to Tech button is clicked, the line’s status will be changed from Pending (or the statuses defined via registry: see Global_Settings_Release_To_Tech_Statuses ) to the Default Status for the technician. This functionality is designed for use with TechAnywhere. A work order can be created as pending, but not be sent to the technician’s wireless device until is released by the dispatcher. When this option is implemented, all new wo’s for technicians with remote access should be created with a status of Pending.
