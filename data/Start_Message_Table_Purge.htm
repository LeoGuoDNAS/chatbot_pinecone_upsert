Start Message Table Purge/Archive

Purpose:

This function is designed to clean up the Message table (used by TechAnywhere, SaqeQuest, FMMS, etc.) and improve the performance of the Work Order TechAnywhere tab. By default, this function will purge / archive all Posted messages that are 60+ days old. Unless disabled via registry, these messages will be archived to the new mssgearcve table.

When this function is selected, it runs without further user input. The intention is that this purge/archive can be run unattended (like the other purge/archive functions) as a Scheduled Event.

Registry:

See Registry Start Message Table Purge/Archive for details on configuring this function.
