Registry Company Address Entry (companyaddress)

Purpose:

This registry entry is used to setup the company address (optionally, by Branch and User Id) for certain system reports including W2's, 1099’s and construction invoices (sets the remit to address on rininv3).

Intercompany Users: 
In Version 9.1, if Intercompany processing is enabled then the W2 Employer Info will now come from the appropriate Company records, NOT the registries:

CompanyName

Company Address

CompanyFederal EIN

Version 9.1 also supports the use of the Company Record to supercede this registry on the following updated reports:

Report AP 1099's (RAP1099v82b01 dated 2019-12-31 or later)

Report AP 1099 Electronic File (RAP1099EF dated 2020-01-08 or later)

Report W2 Electronic File (RW2EFv72b105 dated 2020-01-08 or later)

Report StateUnemployment Electronic File (RSTTEUE dated 2020-01-08 or later)

 

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Setting Values are Uppercase-Lowercase Sensitive:
The Setting Value is case-sensitive: be sure that you enter the value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.

Important Fields:

Id:

companyaddress-branchid-userid

companyaddress--userid

companyaddress

If W2's are reported by Branch, branchid is replaced by the Branch Id this company address is for. If this entry is specific to a user, the branch must be specified if a user belongs to a branch. If branches are not implemented, two dashes (not separated by a space) must precede the user id.

If the company address should be the same for all users, the last format (companyaddress) should be used for the Id.

Setting Key: line1 Setting Value: address1

Setting Key: line2 Setting Value: address2

Setting Key: line3 Setting Value: address3

Setting Key: line4 Setting Value: address4

Setting Key: line5 Setting Value: address5

where address1-5 are replaced by the address lines as you wish them to be printed on various reports. Note that you need only specify as many address lines as needed.

Example

Setting Key: line1 Setting Value: 9450 Midwest Ave.

Setting Key: line2  Setting Value: Garfield Hts, OH 44125

Example for Printing the Telephone Number on Vendor 1099’s

The system used this registry entry to print your firm’s address on the Vendor 1099 report. If you wish to include your firm’s phone number on this report, you may wish to temporarily change this registry just before you print the 1099 (make sure that you save the registry entry before you run the report):

Setting Key: Line1 Setting Value: 9450 Midwest Ave.

Setting Key: Line2 Setting Value: Garfield Hts, OH 44125

Setting Key: Line3

Setting Key: Line4 Setting Value: 216-663-5600

After your 1099’s were printed, you would edit the registry entry so that your phone number would NOT print on your other reports:

Setting Key: Line1 Setting Value: 9450 Midwest Ave.

Setting Key: Line2 Setting Value: Garfield Hts, OH 44125

Setting Key: Line3

Setting Key: Line4 Setting Value:
