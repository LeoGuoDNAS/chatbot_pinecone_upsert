Print List (Tabular Report) Button

The Print List button enables you to print the listing currently displayed in the viewer if the Export box is selected for this Viewer in User Groups and Functions. To print a report:

1. Select the desired list order in the selector pane at the bottom of the viewer.

2. If the filter button appears on the top button bar, you can use it to define the range of records that you want to appear on the listing.

You may reorder the report as desired by clicking on a column title. The report will mirror what appears on the list view when the report is run.

3. Click the Print List button. When the Print Setup window appears, click on OK to continue.

4. Your report will be displayed in a Report viewer. At this point you can page through the report, print it by clicking the Print button on the side button bar, save it to disk, or e-mail it.
