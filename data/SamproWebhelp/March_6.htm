DBFleet Updates

DBFleet  UI Update - March 6, 2020

  • The DBFleet UI has been updated, including adding a New Release page to keep you informed of new changes!

  • You can review this page under the “Release Notes” option in the updated menu.

  • We’ve moved the button to the top right corner of the page, it can be opened by selecting the purple 3x3 icon. Please note that we have updated the name of the “Vehicles” menu to “Devices”

Simplified Display Panel


Pinned Menu

Previously, Work Orders, Pinned Items, and Filters were in three separate menus. We’ve combined them into one panel on the left side of the screen.


Vehicles
We’ve condensed the view of the Vehicle Info to allow more vehicles to display on the screen. The same data is shown, but we have moved the current speed and speed limit to the right of the info card.

 

Techs - NEW For TechAnywhere Users
Users of the TechAnywhere-DBFleet integration will see a new menu on the screen. TechAnywhere Devices display just like vehicles, but info is kept in a separate menu. If you are interested in adding the integration, please reach out to fleetsupport@databasics.com

 

Work Orders
The Work Order Info has also been condensed to fit more on the screen!

 


Filters
Work Order Filters and Fleet Groups can be found in this menu. Update which work orders appear on the map with filters, and which technicians appear using FleetGroups!

Pinned
Pinned Work Orders, Vehicles, and TechAnywhere Devices are added to a menu on the right. Click the updated Pin icon to save the info card to the Pinned Menu. It will always display

 

 

Other Updates

  • Custom Date Ranges have been added to the Work Order Filters

  • If a Technician was in multiple vehicles in one day, the Daily Route View will now display each vehicle, you can select which vehicle's routes you wish to view.
