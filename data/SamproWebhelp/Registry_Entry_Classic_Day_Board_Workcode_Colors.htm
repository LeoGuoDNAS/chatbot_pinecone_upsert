Registry Dispatch Classic/Day Board Workcode Colors (sys-dispatchboard-colors-workcode)

Purpose:

This registry entry controls background color of the Work Code field on default Classic and Day Board views based on the color you assign it.

Usage:

Work Codes that are not specified in this entry will retain the background color of the cell; this may be set by the Technician Status as specified in the sys-dispatchboard-colors-status entry.

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly. You will need to close and reopen the Dispatch Board to see this registry take effect.

Important Fields:

Id:

 sys-dispatchboard-colors-workcode

Specify Work Code Colors

 Setting Key: workcodeid Setting Value: color:Custom

where workcodeid is replaced by a valid work code id, and color is replaced by the Windows color map index.

Important Note: all registry options that use the color functionality must be correctly configured or a system primitive error will appear when the Dispatch Board is opened. These entries will NOT be ignored if an ‘x’ is placed in from of them to temporarily disable them.

A. Verify that the registry entry has been saved.

B. To enter the index number, enter all information up to the word Custom. For example, to set the background color for HVAC, you would type HVAC=color:

C. After entering the colon: double-click on the line to display the Windows color selector palette, click on the desired color and then click OK.
