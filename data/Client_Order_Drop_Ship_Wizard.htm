Client Order Drop Ship Wizard

Purpose:

When a client orders an item that you do not have in stock, one solution is to have it drop shipped from your vendor directly to your customer. The Wizard is designed to step you though the process of completing the Client Order and Generating a Purchase Order for the merchandise. The COP Invoice for this order will be held:

1. Until the Purchase Order is vouchered.
OR
2. The items are "received" in the Inventory Receipt Journal (to allow the invoice to be produced sooner). This option required a registry entry (Global Settings Registry to Receive Dropshipped COP's in Inventory Receipts and Allow Billing).

The Drop Ship Wizard will only be invoked if the order type is ‘order’. This implies that you can save an order with drop shipments as a ‘quote’ without invoking the wizard. When the quote becomes an order, the operator will change the order type from ‘quote’ to ‘order’. The Drop Ship Wizard will then be invoked when the order is saved.

Usage:

If the Vendor's No Inventory? field is checked, the Vendor may not be specified when running this function.

The Drop Ship Agent box in the user’s record must be flagged to enable drop ship processing.

Tip: Customer Changes Order
Assume that your customer calls up and wants some material. You create a new Client Order, add 20 items, all of which are to be drop shipped. You create the PO through the wizard. ½ Hour later customer calls up and says that they won’t need 5 of those items.

To correct this, you can't delete the line off the Client Order until you delete it off the PO (which it does allow). Then the line(s) will delete.

Important Fields: Verify Purchase Order Lines

Vendor Id:
Enter the Vendor that should drop ship this item, or double click to select from a list. If a Primary Vendor is specified on the Inventory Item’s Purchasing tab, it will default for you. To change the Vendor, or to specify one if it did not default, double-click on the Vendor Id to select from a list, or enter the Vendor’s Id from the keyboard.

Inventory Id: Inventory Item Name:
This display only fields defaults from the Items on the COP Order.

Vendor Part Number: Unit Cost:
If the selected Vendor has their own part number associated with this item, it will be listed.  In addition, if this item has been purchased from this Vendor, the most recent purchase price will be listed. Both fields may be updated if needed.

Quantity:
This field defaults from the Client Order; the user may adjust it.

Important Buttons:

Click the Cancel button to stop the generation of the drop ship purchase orders.

Click the Finish button to generate the Purchase Order; the system will display the PO Id when the generation is completed.
