Registry Start NetFramework Post

Registry Id: sys-netframework

Setting Key: UploadFolder  Setting Value: Absolute file path

The UploadFolder setting is the default location where files from SAMNetFramework applications (e.g., SubAnywhere) are uploaded.  This setting can be overridden by the registry of specific SAMNetFramework applications (note for Programmers: for Sntfrmewrkpst to properly retrieve the file from the overridden location, "Application" must be specified in Message record contents).
