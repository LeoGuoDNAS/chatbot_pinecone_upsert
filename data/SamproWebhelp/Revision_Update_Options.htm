Revision Update Options

Purpose:

When you update to a new Data-Basics release, the software will check the revision level of your database. If your database must be updated for the new release, the Revision Updates Options screen will be displayed.

Usage:

This screen enables you to implement an online backup prior to updating your software (highly recommended!) and to set the default directories for the backup and the database unloads.

Important Fields:

Online Backup?
If this box is checked, your database will automatically be backed up before and after it is converted. It is highly recommended that you do so, even if you have been running other forms of backup (such as a tape drive). A power outage or hardware glitch during conversion could case irreparable damage to your database.

Backup Directory:
By default, the Backup Directory will be set to a subdirectory of the directory containing your executable file (v.exe). You may change the path to the drive and directory of your choice (e.g., D:\databasics). The directory path that you specify should not include blank spaces.

Unload Directory:
By default, the Unload Directory will be set to a subdirectory of the directory containing your executable file (v.exe). You may change the path to the drive and directory of your choice (e.g., D:\databasics). The directory path that you specify should not include blank spaces.

Click the Go button to initiate the backup and conversion.
