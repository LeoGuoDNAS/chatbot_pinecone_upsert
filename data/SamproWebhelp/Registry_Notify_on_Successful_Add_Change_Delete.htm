Registry Notify on Successful Add, Change, Delete

Purpose:

The software disables keyboard and mouse-driven actions in SAM Pro while record viewers are saving data. The following optional registry entries may be used to display a message when a record has been successfully added, saved or deleted (note that the default is not to display the message).

Usage:

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Setting Values are Uppercase-Lowercase Sensitive:
The Setting Value is case-sensitive: be sure that you enter the value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.

Important Fields:

Id: sys-viewer-afunctionid

Key Setting: NotifySuccessfulRecordChange Key Value: true

Key Setting: NotifySuccessfulRecordAdd  Key Value: true

Key Setting: NotifySuccessfulRecordDelete Key Value: true

 

Example: Notify User when Work Order Save is Complete

Id: sys-viewer-vwrkordr

Key Setting: NotifySuccessfulRecordChange Key Value: true
