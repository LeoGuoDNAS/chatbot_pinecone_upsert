AP Journal Payments Tab

Purpose:

The information on this screen displays payment information directly from the AP Journal for paid Vendor Invoices.

Important Fields:

Double-click a row to open the Check viewers. If the Check Reference Type is V then it opens View Check Vendor, if it's blank then it just opens the regular View Check, Reference Type Employee was filtered out in the query.

Payment Status Bar:

TrxId: Displays the Transaction Id

Vendor: Displays the company name of the vendor for the paid invoice

Invoice: Displays the invoice number. 

Payment Details:
Check ID, Check #, Date, Payee Name, Amount, and Vendor populate from the AP Journal

If a check(s) has been generated for this vendor invoice, launch and filter the View Check or View Check (Vendor) for this entry. 

Users must have access to View Check or View Check (Vendor) for this button to appear (or be functional).
