Start TechAnywhere Post

Purpose:

This function processes completed work orders that have been uploaded from TechAnywhere. Note TechAnywhere status changes and new work orders are processed by Scheduled Event Manager directly and do not require this function to be initiated.

Usage:

The user may initiate this function by selecting Start TechAnywhere Post and clicking the Go button. Typically, however, it will run as a scheduled event on a regular basis.

Uploaded collateral was previously attached to the Site, Equipment and Work Order. Now, if the Document Category equals 'Receipt' or 'Invoice', these documents will only be attached to the Work Order.
