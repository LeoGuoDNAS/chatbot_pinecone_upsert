Service Evaluation Wizard – Version 7.2 and Earlier - noprint

Purpose:

The Service Evaluation Wizard will be launched when a work order that references a Service Evaluation table is completed or invoiced (depending upon the category of the questions being asked). The Evaluation questions will only be asked the first time the work order escalation status is set to completed or ready to invoice.

The responses will be written to the Evaluation tab of the work order, and can be used to generate a numerical grade for the primary technician.

Usage:

References the Service Evaluation Viewer , Service Evaluation Table Viewer .

CLS Customization Only – version 7.0

When running under the CLS customization only, service evaluations that would normally be triggered when the work order escalation status is set to Invoice Ready will instead pop when the work order escalation status is set to Invoice Review.

Important Fields:

Evaluation Question: Goal:
The question you are answering (and the goal of the question) default from the Service Evaluation Table referenced in this Work Order.

Response:
Double-click on this field to select from a list of valid responses, or simply enter a valid reply.

Explanation:
This optional field enables you to provide a brief explanation of the answer.

Important Buttons:

Click the Finish button when all questions have been answered; the responses will be written to the work order’s Evaluation tab.
