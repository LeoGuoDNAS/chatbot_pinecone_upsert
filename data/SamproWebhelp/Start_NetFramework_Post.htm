Start NetFramework Post

This function provides a SubAnywhere Scheduled Event Manager for Document Uploads.  Specifically, it handles messages for SAMNetFramework applications such as SubAnywhere.  Currently, this function supports handling Document Uploads in a fashion similar to DashboardEventManager and Start Dashboard Post.  This function should be implemented by add the Start NetFramework Post to a scheduled event to be run at an interval of your choosing.

See sys-netframework for a required registry entry to define the upload folder.
