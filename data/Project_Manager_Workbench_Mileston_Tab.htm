Project Manager Workbench Milestone Tab

Purpose:

This tab associates Milestones with this Job record. Milestones are used for reporting purposes. Changes made to this tab will be reflected on the Job viewer.

Usage:

See Project Manager Workbench for details on this viewer’s upper tabs.

See Milestone Viewer for details on the Milestone record.

Important Fields:

Milestone Id:
Enter a Milestone Id or double-click to select from a list.

Start Date: End Date:
The Starting and Ending dates for this Milestone.

Description:
A brief description of this Milestone.

Completed?
Check this field (by double-clicking on it) if this Milestone has been completed. When you do so, your User Id and the Date Marked will be displayed by the system.
