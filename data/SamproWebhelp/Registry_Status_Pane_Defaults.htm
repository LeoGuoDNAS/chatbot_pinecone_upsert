Registry Status Pane Defaults

Purpose:

The following registry entry may be used to adjust the status line’s height, foreground color, background color and font.

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Setting Values are Uppercase-Lowercase Sensitive:
The Setting Value is case-sensitive: be sure that you enter the value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.

Important Fields:

Id:

sys-statuspane-branchid-userid

sys-statuspane--userid

sys-statuspane-branchid

sys-statuspane-securityid

sys-statuspane-usergroupid

sys-statuspane

Hint: you may use the security option of this entry to help remind multiple security users how they are logged in.

Specify Font Parameters

Setting Key: Facename Setting Value: aFaceName

where aFaceName is replaced by a font name (e.g., Arial, Tahoma). Note that font must be installed on the user’s computer to be displayed.

Setting Key: bold Setting Value: yes or no

Specify yes to use the bold version of the font, or no to use the normal version.

Setting Key: points Setting Value: aPointSize

This option specifies the size in points. Point sizes ranging from 11 -14 are recommended

Specify Height of Status Bar

Setting Key: height Setting Value: #ofpixels

This option specifies the height of the status pane in pixels. As a general rule of thumb, the height in pixels should be at least double the point size. For example, if your font is set to 11 points, the height should be set to 23.

Specify Status Line Colors

Setting Key: backcolor Setting Value: {double-click and select a color from the palette}

Setting Key: forecolor Setting Value: {double-click and select a color from the palette}

The backcolor option sets the background color of the status pane. The forecolor option set the color of the text.
