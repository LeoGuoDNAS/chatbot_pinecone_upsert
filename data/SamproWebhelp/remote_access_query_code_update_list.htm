Remote Access Query: code_update_list

Purpose:

Specifies the software files what will be uploaded to the TechAnywhere wireless devices during the update.

Usage:

Currently, only javascript (.js) and secure browser (.hta) files are supported in the update.

Also see Remote Access Query: next_techanywhere_versions_list for other important information on updating.

Important Fields:

Application: TechAnywhere

Dataset Name: code_update_list

Query Id: This field is assigned by the system when the record is saved.

Query Format: Frame

DataSet Query:

[

"file1.js",

"file2.js",

"file3.hta"

]

 

Replace file1.js and so on with the actual names of the files to be updated. Note that only javascript (.js) and secure browser (.hta) files are currently supported for code updates.
