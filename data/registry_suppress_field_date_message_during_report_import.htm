Registry Suppress Field Date Message During Report Import

Registry Option to Suppress Field Date message during import

Id: sys-ffv-global-settings

Setting Key: ReportImportReportFields

Setting Value: Keep or Replace

If this option is absent, the system will prompt you whenever an imported report field has an older date than the current value in the database.

If this option is set to Replace, the message will be suppressed, and the system will always replace the report field using the report being imported: this is the preferred setting for client installations.

If this option is set to Keep, fields will NOT be updated with newer ones from the report. This setting would only be appropriate for a report writer.
