504 Flat Rate Pricing Implemented

TechAnywhere now supports Flat Rate Pricing on COD Work Orders only. In order to be considered a COD Work Order, the Job associated with the Work Order must reference an AR Terms record that has the COD box checked.

If this option is implemented, the TechAnywhere user will be able to import Flat Rates during synchronization. A Flat Rate Id may then be selected in on the Flat Rate TechAnywhere page (accessible from the TechAnywhere Work Order Detail page for COD Work Orders only).

When a Flat Rate Id is select in TechAnywhere, the pricing information only will be imported into TechAnywhere.  The actual usage of both time and materials must be entered manually by the tech.
