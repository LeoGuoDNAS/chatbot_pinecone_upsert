Registry Opportunity (sys-viewer-vopprtnty)

Purpose:

The following registry setting is required to allow you to create Client Sites from the CRMAnywhere Dashboard or the SAMPro Opportunity viewer.

Related Registries:

See Registry CRM Opportunity Templates for defining Opportunity records to use as Templates in CRMAnywhere.

Important Fields:

Id: sys-viewer-vopprtnty

Setting: NewSiteTemplateId    Setting Key: Site Template Id

The template site should be very basic since it must function for all Sites generated from Opportunities in SAMPro or CRMAnywhere. The Client Site Template should have its Quote Defaults tab completed.  It must also have a Site Equipment record associated with it.
