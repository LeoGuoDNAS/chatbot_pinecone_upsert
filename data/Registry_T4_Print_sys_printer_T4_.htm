Registry T4 Print (sys-printer-T4)

Purpose:

This registry entry specifies any margin adjustments needed for alignment.

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Setting Values are Uppercase-Lowercase Sensitive:
The Setting Value is case-sensitive: be sure that you enter the value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.

Important Note: Printer Requirements

Unless you are using the new Report PR Canadian T4 (Standard) report, your T4 printer must support native PCL printing: host based or driver-based PCL is not acceptable due to the need to print exact output to the T4 forms.

Additional Registry Entries for T4 Printing

In addition to this registry entry, Federal Business Number must be specified in the CompanyEINFederal registry entry. Required default values for this report must be made in Registry T4 Report Defaults .

The registry options discussed in Registry T4 Print (sys-printer-T4) will most likely be required to alignment this report to print properly. The Registry T4 Summary Print adjusts the T4 Summary.

Company Record Support

In SAMPro Version 9.1.611.054a, the Canadian T4 reports have been revised to use the Company record / Company Registry tab to specify the Company Name and Address and other registry settings required for these reports.

Company Name, Address: These reports will use the Company Name & Company Address from the Company Record Header info if available.

Company Registry Tab Fields: Fields not defined in the Company Record header may be added to the Company record's Registry Tab.  For example, the t4sin1 for the Summary report.

First look for the Company Record's Registry entry. If not found,
Second look for the sys-report-rT4 If the registry is blank or not found,
Third look for the company specific companyname registry setting (e.g., companyname-1) and use that.  
If none of the above are available, the string 'companyname' will be used in the report instead of leaving it blank.

Important Fields:

This registry entry specifies which printer (named in the Contents field) T4's are sent to.

Id:

sys-printer-T4-branchid-user

sys-printer-T4-securityid

sys-printer-T4

If the optional branch and user id's are specified in the Id, only T4's for this branch (and user if specified) will be sent to the printer. If this entry is specific to a user, the branch must be specified if a user belongs to a branch. If branches are not implemented, two dashes (not separated by a space) must precede the user id.

Name:
Briefly describes this entry (e.g., HP 4000 T4 Printer).

Adjust the Margins

Setting Key: marginx Setting Value: n.n

Setting Key: marginy Setting Value: n.n

Setting Key: marginybottomform Setting Value: 14.0

These optional formats are used for alignment purposes. Note that #.# is entered in centimeters, and is used to specify offset from the "home" position (the upper left-hand corner of the page).

Example for Report Canadian T4s (Standard)

Tip:  Most printers require negative margin adjustments for the new Report PR Canadian T4s (Standard).  We recommend starting with the following settings and adjust as needed following your alignment print:

Id: sys-printer-t4
Setting Key: marginx  Setting Value: -0.63
Setting Key: marginy  Setting Value:  -0.85

Reminder: Negative margin setting values move the report text closer to the upper left-hand corner.  Setting values without the negative sign move the report text to the right (marginx) and down (marginy).

Specify the T4 Printer (not needed for the new Report PR Canadian T4s (Standard)

Setting Key: printer Setting Value: printername

where printername is the name of the printer exactly as it is displayed in the printer dialog box that appears when you select a printer in SAM Pro. Please note that this field is upper/lower case sensitive. Also, if network pathing is included on the printer name, make sure that the backslashes are correctly entered.

: to move things right marginx=##’ or left use: -#.#

: to move things down use ‘;marginy=#.#’ or up use: -#.# 

 

Example with Network Pathing, output moved to the left and down:

Setting Key: Printer Setting Value: \\myownserver\Samsung ML-3050 Series

Setting Key: marginx Setting Value: -0.3

Setting Key: marginy Setting Value: 0.1

Setting Key: marginybottomform Setting Value: 14.0

Example without Network Pathing, output moved to the right and up:

Setting Key: Printer Setting Value: Samsung ML-3050 Series

Setting Key: marginx Setting Value: 0.3

Setting Key: marginy Setting Value: -0.1

Setting Key: marginybottomform Setting Value: 14.0
