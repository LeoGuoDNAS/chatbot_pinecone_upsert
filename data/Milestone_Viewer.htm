Milestone Viewer

Purpose:

This record enables you to define 'milestones' for reporting purposes and use in DBAnalytics. This record is referenced in the following records:  

Jobs: Typical Milestones pertaining to Jobs could be Preliminary Meeting, Budget Review, etc.

Opportunities: Typical Milestones for Opportunities could include: Pre-Sales, Initial Customer Meeting, Create Quote, Submit Quote, Close Sale.

CRM: Access from the Opportunities record.

Work Orders: The Work Order's Milestones Tab is auto-populated by the Milestone Group selected in the Start Project Work Order Generator function.

Important Fields:

Milestone Id:
This required field uniquely identifies the Milestone.

Milestone Description:
This field briefly describes the Milestone.

Obsolete?
If this field is checked, the record is considered Obsolete, and it may not be directly specified during data-entry (either by entering the Id or selecting it from a list). Obsoleted records can be edited and the Obsolete option cleared when necessary. When you check the Obsolete field, the following message box will be displayed:

“You have elected to obsolete this record. Would you like to see existing references to this record?”

If you click Yes, the system will display a viewer that provides click-through access to all tables referencing this record: see References List for details.
