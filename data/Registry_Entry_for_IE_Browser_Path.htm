Registry Internet Explorer Browser Path (iebrowserpath in sys-ffv-global-settings)

Purpose:

This registry entry identifies the path to the Internet Explorer execution file (IEXPLORE.EXE). This registry entry is only required if Report Anywhere is implemented on your system, you previewer reports as HTML, and Internet Explorer is loaded to a location that is not recognized by the system.

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Setting Value Notes:
The Setting Value is case-sensitive: be sure that you enter the value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.
The equal sign = is a reserved character in the Setting Value.  It should NEVER be used in the Setting Value field unless you are specifically instructed to do so for particular registry setting.

Important Fields:

Id: sys-ffv-global-settings

Setting Key: IEBrowserPath  Setting Value: path

where path is replaced by full path to the directory containing your internet explorer execution file, IEXPLORE.EXE (e.g., C:\Program Files\Internet Explorer).

Hint: After saving this entry, remember to exit the software. When you log on again, your browser path entry (entries) will be in effect.
