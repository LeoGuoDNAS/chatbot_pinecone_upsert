Filter Closed Button (Show Inactive/Dead Records)

Click this button to display only those records with a status of inactive (terminated, closed, etc.) and a closed date of earlier than today. Records that are flagged as Obsolete will also be displayed on the Closed list.

In journal viewers, only those final journal entries made to inactive accounting/payroll periods will be displayed. Note that the Next Id of these journal entries is blank.

Note that Deleted Journal entries will not appear on the closed listing because their Next Id is set to ‘Deleted’. To view only Deleted entries, enter Deleted in the Next Id range, clear the other ranges, and click the Filter Records button to display the list.
