Vendor Type Viewer

Purpose:

Vendor Types may be defined for categorizing your vendors for reporting purposes.

Usage:

Reference in the Vendor record.

Example:

Subs, Parts, Other.

Important Fields:

Vendor Type Id:
Uniquely identifies this record. This field is required.

Description:
Briefly describes this type of vendor. Optional but highly recommended.

Obsolete?
If this field is checked, the record is considered Obsolete, and it may not be directly specified during data-entry (either by entering the Id or selecting it from a list). Obsoleted records can be edited and the Obsolete option can be cleared when necessary.
