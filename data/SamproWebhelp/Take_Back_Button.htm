Return Id (Take Back) Button

This button is used when you have opened a viewer from an Id field in order to insert information into a data entry field in the parent window. To do so, click on the item ID you wish to pull back to the parent viewer, and then click the Return button. Â 

Alternately, you can take an ID back to the parent viewer by double-clicking on that ID with the right mouse button.
