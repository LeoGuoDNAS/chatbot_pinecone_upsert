Service Evaluation Viewer

Purpose:

Defines the individual questions that will be asked when the Unanswered Service Evaluation Wizard is run when a Work Order is completed, invoiced or achieved a registry-defined Work Order Escalation Status. The responses will be written to the Evaluation tab of the work order, and can be used to generate a numerical grade for the primary technician.

Usage:

Referenced by the Service Evaluation Table Viewer, and the Unanswered Service Evaluation Wizard. Service Evaluations are run for the primary work order technician.

Registry:

See Work Order Registry - Service Evaluations Option for details defining the Work Order Escalation Status Changes used to invoke the Unanswered Service Evaluation Wizard.

Important Fields:

Service Evaluation Id:
Uniquely defines this record.

Evaluation Question:
The question that will be asked when the Service Evaluation Wizard is run when a work order is closed or invoiced. This question must be formed so that it can be answered with a Yes to indicate a successful outcome; only questions answered ‘Yes’ will earn evaluation points. Zero points will be awarded to questions answered ‘No’ or ‘N/A’.

 Example:

Correctly formed question: Was the work performed on time?
Incorrectly formed question: Was the work completed late?

Category: o Closing o Invoicing o General
By default, this field is set to Closing: this question will be asked when a work order is closed. To display the question when the work order is invoiced, select Invoicing by double-clicking the adjacent radio button.

The General option allows the inclusion of general questions to be included in Service Evaluations. The Unanswered Service Evaluation Questions Wizard will be invoked when the Work Order Escalation Status is set to the value (or one of the values) defined in the sys-viewer-vwrkordr registry’s GeneralEvaluationLogTypes option. Note that all unanswered questions (including Closings and Invoicing questions) will be displayed in the wizard when it is invoked in this fashion.

Yes Response: Acceptable?
Check this box if ‘Yes’ is an acceptable reply for this question. Typically, this box would always be checked. When computing the Technician Grade, points will be added to the total points available and total points earned for answering Yes.

No Response: Acceptable?
Check this box if ‘No’ is an acceptable reply for this question. Typically, either this box and/or the N/A would be checked. When computing the Technician Grade, points will be only be added to points available (not points earned) for an answer of No.

N/A Response: Acceptable?
Check this box if ‘N/A’ is an acceptable reply for this question. Typically, either this box and/or the No would be checked. When computing the Technician Grade, no points will be added to either the total points available or the total points earned for answering N/A.

Applies to Employee?
This question will only be displayed if an Employee Id is referenced in the record of the Technician being evaluated.

Applies to Employee?
The Service Evaluation Wizard will only display this question if an Employee Id is referenced in the record of the Technician being evaluated.

Applies to Vendor?
The Service Evaluation Wizard will only display this question if an Employee Id is referenced in the record of the Technician being evaluated.
