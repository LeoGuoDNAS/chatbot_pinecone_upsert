Tech List (Change Dispatch Board Button)

Version 7.3+:

The Tech List button enables you to choose a different list of Techs for this dispatch board. This feature allows each dispatcher to select their techs, or for a single dispatcher to switch boards quickly.

When you push the Tech List button, a list of available boards will be displayed. Double-click on the board that you wish to display with the left mouse button.

The Board Technicians Viewer may be used to define the lists of technicians that appear on the dispatch board.
