Report PR Certified Payroll - PDF

Purpose:

This report allows the user to automatically complete the WH-347 fillable form from the Department of Labor for weekly payroll. If the report is not 'flattened' via registry, the user will be able to update the PDF as needed.

Usage:

A check must have been issued for an employee's hours to appear on this report.

Registry and Setup:

See Registry PR Certified Payroll - PDF for instructions on setting up this report.

Range Screen:

Job Id:
This field must be completed so that the report is run 1 job at a time.

Ending Date:
Enter the "end date" from the Payroll Period you wish to report on.

Form Style: Federal wh347
This report currently supports the Federal WH 347 report.

The SAMPro base report will initially be displayed: click the Fillable Form button to launch the PDF. Unless auto "flattened" per registry setting, the resultant file may revised as a Fillable Form and then saved or flattened once finalized.
