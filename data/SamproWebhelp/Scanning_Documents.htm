Scanning Documents

SAM Pro supports document scanning and archiving. The scanned document will be saved as a .jpg image (or a PDF if a registry entry is enabled). Multiple pages are supported. This functionality may be used in a variety of purposes, such as scanning in vendor invoices as they are vouchered in AP.

Registry:

See Registry Entry Global Settings: Save Scanned Docs As PDF .

Usage:

SAMPRO scanning technology attaches documents to the main record being scanned as well as associated records the following special cases:

AP Journal

Vendor

Purchase Order (if specified)

Job (all jobs referenced on any child row)

Work Order (all work orders referenced on any child row)

Labor Journal

Employee

Payroll Period

Job (all jobs referenced on any child row)

Work Order (all work orders referenced on any child row)

AR Receipts Journal

Client

Job (all jobs referenced on any child row)

Invoice Master (all invoices referenced on any child row)

Setting Up

1. Extract the compressed file samproscan.zip into the directory containing the SAM Pro executable file (v.exe). samproscan.zip contains the program acquire.exe that SAM Pro will call to acquire the scanned image.

2. All users that will be scanning documents must install a local scanner with TWAIN compatible driver software. It should be set up to scan in 256 grayscale or full color.

TIP: Setting the Twain Scanner as the Default
SAMPro will always pick the last used scanner. So what you may need to do to change your default/last used scanner is to open up a program like Microsoft Word, and use the built in scan function (insert picture from scan), pick which scanner you want to use, perform a scan, and then start up SAMPro. SAMPro should then use the scanner that you picked in MS Word.

3. Create a directory on the local machine in which the scanned images will reside.

4. To enable scanning, you must have a registry entry for document archiving with an archive directory set. See Registry Entry for Document Attachment & Scanning for details.

Scanning a Document

1. Access the SAM Pro record that you want to associate the scanned document with. For example, if you wish to scan the vendor invoice you have just received, the AP Journal entry you are using to voucher that invoice should be displayed.

2. Verify that the scanner is turned on, and that scanning is enabled in the record window. The record must be previously saved with no changed for scanning to be enabled!

3. Click the Scan Document button. The Scan Document Wizard window appears instructing you to ‘’Load Next Document to Scan. Insert the document you wish to scan in the scanner and then click the Scan Page button.

4. The Wizard will ask ‘Did Current Document Scan Ok’. After the scanner acquires the image and the Transferring Data box disappears, click the Yes button. If the Transferring Data box did not appear and the scanner did not seem to scan the document, click no and verify that the scanner is operational.

5. The Scan Document Wizard will prompt you to load the next page to scan. If you are scanning a multi-page document for this record, click Scan Page and repeat the scanning process. If you have scanned all pages for this record, click Finish.

The Scan Wizard displays a warning message if the user presses the Cancel button. At this point, they may confirm the cancellation or return to the wizard to finish the scan.

6. A final screen will be displayed that allows the user to enter the Document’s Description and Category. The user should press Done after entering these fields (if applicable. The Document Scanning Wizard now also allows the user to set the 'viewable' options for the document.

The scanned page(s) are now attached to the record(s). They may be viewed by clicking the Attachment button located just above the Scan Document button. Note that multiple pages may be viewed from the ‘OtherFiles’ tab of the Document Viewer.

PDF Enhancement for Documents with Multiple Images
If the file in the Document Viewer header is a bmp, jpg or gif AND the Document contains multiple images, the Show Document button (in the Document header) and the View Document Button (on the Document List) now offer the user the option of displaying the images as a single PDF file.  At this time, only bmp, gif and jpg files will be assembled into the pdf.

Files Originally Scanned as PDF’s:
By default, the Scan Document Wizard saves the scanned documents as .jpg files. If you opt to save scanned documents as .pdf files via registry, each page will be saved as a separate .pdf file. Additional pages must be accessed from the Other Files tab of the Document Viewer: individual .pdf files will not be combined for viewing like .jpg and .bmp files.
