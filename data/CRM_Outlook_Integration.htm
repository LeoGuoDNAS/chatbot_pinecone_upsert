CRM Outlook Integration

 

The following steps will need to be completed on each desktop/for each user that wishes to utilize the CRM Outlook calendar integration.

1.       Copy the following folder and files to a local directory (create a temporary folder on your desktop for example):

2.       Once the files are on your machine, execute the setup.exe file.

3.       If Outlook is running, close and restart the application. Upon restart you will see a CRMAnywhere icon in the top tool bar of the calendar section of Outlook (far right).

 

4.       Click on CRMAnywhere, then configure. You will fill in your DBAnalytics URL as well as your SAMPro User Id and password and save.

 

  

 

 

 

 

5.       Next go to Manage Calendars from the CRMAnywhere drop down, select your User Id from the list and save.

6.       Your CRM calendar will be available for selection on the left-hand side of Outlook. There is also a CRM Group Calendar that will show all activities for any other users you have selected from the Manage Calendars section of the CRMAnywhere dropdown (from step 5).

 

Any activities displayed on your DBA Calendar analytic will now be synced to your Outlook calendar. Use the Refresh Calendars option from the CRMAnywhere drop down (top tool bar) if you do not see an anticipated event. To edit the event from Outlook - right click on the event and select CRMAnywhere Edit Event. This is the only way changes made from Outlook will be reflected in CRMAnywhere.

                                                           

 

Troubleshooting

You may need to enable add-ins in your Outlook settings for this to work and/or work efficiently.

File > Info > Manage COM Add-ins > CRMOutlookAddIn2013_2016 > Options > Always enable
