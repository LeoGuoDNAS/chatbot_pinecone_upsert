About this Product

If you select the Help menu and choose the About this Product option, the system will display an information box listing the SAM Pro version number, date, and database revision level. If you choose the About this Product option from a UD Report window, the Report Id and Revision Level of the report will also be displayed. Note that system (Smalltalk) reports do not contain this information.
