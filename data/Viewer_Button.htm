Viewer Button

 
The Viewer button displays all record viewers available to your user group.

 
Pressing the Starter button displays the list of all application starter functions available to your user group.


The Desktop button displays your custom desktop if one has been defined for your user account.

 
Pressing the Menu button displays the list of all application record viewers, starter functions, and reports available to your user group. This button is generally located on the lower half of the scrollable top button bar.

 
The Report button displays all reports available to your user group.
