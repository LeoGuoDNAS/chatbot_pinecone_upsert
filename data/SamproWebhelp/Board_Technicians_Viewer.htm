Board Technicians Viewer

Purpose:

Set up the dispatch boards and assign technicians to them.

Example:

Suppose that your company employs two dispatchers - one for HVAC and one for plumbing. In this case, you could create two dispatch boards (HVAC and plumbing), assigning your HVAC technicians to the HVAC board, and your plumbing technicians to the plumbing board. Since an employee may be assigned to more than one board, you could also create a board that included all employees for days when only one dispatcher was available. Alternately, a dispatcher could simply switch between boards (via the Change Dispatch Board button) from the Dispatch Viewer. This viewer is divided into two sections. The upper section contains the Id and Name fields for this board. The lower section consists of a table that enables you to assign technicians to this board.

Usage:

Required for Dispatching.

All Work Orders Option (Power Board Users)
To display work orders for all technicians, create a Board Technician record with an Id in the upper section but NO technicians listed in the lower section. When this Board Technician record is selected from the Dispatch Board’s Tech List tab, work orders for all technicians (and unassigned work orders) will appear in the Work Order list. The Work Order Summary (in the upper section of the Dispatch Board) will display only two lines – for Unassigned Work Orders and All Work Orders.

Important Fields: Upper Section

Note: this Help Topic lists the standard configuration of all fields that may be included in this record. If you are using a customized system, the fields displayed on your screen may not exactly match the order or status (required, optional, default, display-only, hidden, etc.) of the fields described in this Topic.

Board Id: (15 character, only . and - special characters)
A unique Id that clearly identifies this board (HVAC, All, Jean's Board, Board 1, etc). This field is required.

Board Name: (30 character)
This optional field briefly describes the Board.

Security Id:
If multiple location security is implemented on your centralized database system, this informational field displays the Security Id of the local user that created this record. If multiple location security is not implemented on your system, or if this record was created by a global user, this field will be blank.

Important Fields: Lower Section

Technician Id:
Enter a valid technician Id or double-click to select from a list of technicians. The Technician Name field will be filled in when the Id is entered. Note: Technicians will appear on the Dispatch Board in the order in which they are arranged on this viewer! If you wish them to appear in alphabetical order on the Dispatch Board, be sure to arrange in alphabetical order (by Tech Id) on this screen.
