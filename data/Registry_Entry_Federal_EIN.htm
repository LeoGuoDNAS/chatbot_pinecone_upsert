Registry Federal and State EIN (companyEIN)

Registry Entries Specifying Federal EIN for Companies with/or Without Multiple Branches/Operating Companies

This entry specifies the Company EIN number that prints in box b of the W2 form and for use in the Canadian T4 form (if Intercompany is not enabled in Version 9.1 and above). This registry entry is also required to print the Federal Id on 1099 forms for Intercompany and non-Intercompany users.

Intercompany Users:
If Intercompany processing is enabled then the W2 Employer EIN Info will now come from the appropriate Company records, NOT the registries:

CompanyName
Company Address
CompanyFederal EIN

 

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Setting Values are Uppercase-Lowercase Sensitive:
The Setting Value is case-sensitive: be sure that you enter the value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.

Related Registry Options:

See Registry Entry Vendor 1099 Report to specify a second remit to address line to print on the Vendor 1099.

Optional Registry Entries Specifying State and EIN for Companies with Multiple Branches

This entry specifies the 2-digit State code and Company EIN number that prints in box 15 of the 1099. If the optional branch id is specified, only W2's created for this branch will include this EIN number and state designation. Note that if you do not print W2's by branch, this registry entry is not required: the system will obtain the EIN and State from the Tax Authority record with the Type option "State".

Id:
companyEINstate-st-branchid

specifies the 2-digit State code and Company EIN number that prints in box 15 of the W2 form. When creating this Id on your system, replace st with the appropriate 2-character postal code. If the optional branch id is specified, only W2's created for this branch will include this EIN number and state designation. Note that if you do not print W2's by branch, this registry entry is not required: the system will obtain the EIN and State from the Tax Authority record with the Type option "State".
