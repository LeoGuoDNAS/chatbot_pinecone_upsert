Start Alias Update

Purpose:

The purpose of this function is to copy the credit card Alias information from one Site to other Sites sharing the same Client Id on the Site's COP tab. The Alias, Masked Credit Card Number, and Expiration Date will be copied. Only Sites with an existing Alias AND COP Client are eligible for selection.

Setup Note: Start Function List Update must be run to make this function available from the Starter menu if a data conversion was not triggered with this update. This function would be assigned to specified User Groups.

Initial Range Screen

Site Id: Site Name:
When the Starter is run, the initial screen requires the user to select the Site that contains the Alias to be copied. If this function was launched from the Copy Alias to Sites button on the Client Site Main tab, the Site will be completed for you.

Client Id:  Client Name:
These display fields default from the Client on the Site's COP tab.

Payment Alias - Masked CC Number - CC Expiration Date
These display fields default from the selected Site. This information will be copied to the sites you select on the next checklist screen.

When the user confirms that the correct Site was selected, they should click Go.

Checklist Screen:

Ok - Site Id - Site Name - Payment Alias -  Masked CC Number - Expiration
The checklist of eligible Sites with a matching COP Client displays that Site's current Alias and CC fields.  

The Sites to be updated with the selected Alias (from the previous screen) should have the Ok box(s) checked. Double-click the Ok box to toggle between checked and cleared.

Click the Check button to check (select) every item on the list.

 Click to de-select every eligible item on the list

When Go is clicked from the Checklist the selected Sites will be updated with the revised Alias, Masked CC Number and Expiration Date.
