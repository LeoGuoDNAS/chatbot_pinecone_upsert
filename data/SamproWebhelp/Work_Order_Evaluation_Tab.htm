Work Order Evaluation Tab

Purpose:

If the Evaluation Id field is completed (on the Main tab of this Work Order), the system will generate/complete lines on this tab when the Service Evaluation Wizard is run. The Service Evaluation Wizard is invoked when a work order is completed or invoiced (depending upon the category of the questions being asked). The responses are written to this tab, and can be used to generate a numerical grade for the primary technician by running Start Compute Technician Evaluations.

Technician Evaluation questions are added to the work order based on the Equipment and Tech selected during New Call. Often, the Tech Id changes after the WO is created, but in previous releases the system didn't update the Evaluation lines on the WO so that they were associated with the new Tech.

If the primary Tech changes, the system re-evaluates entries on the Evaluation tab:

Existing lines for the selected Evaluation Table with responses for the previous Primary Tech are retained.

Existing lines with  'unknown' response are deleted for the previous Primary Tech are deleted.  

A complete set of lines for the new primary Tech (as applicable for a Vendor/Employee tech) is added.

Important Fields:

Tech Id:
This field will default to the primary work order technician (on the Main tab) when the Service Evaluation Wizard is run.

Evaluation Id: Evaluation Question: Goal:
Evaluation questions from the Evaluation Table Id specified on the Main tab will be recorded when the Service Evaluation Wizard is run. These fields are sensitive to whether the primary Tech is an Employee or Subcontractor (Vendor).

Response:
The response that was recorded when the Service Evaluation Wizard was run.
