Logon Time Monitoring

Important Notice: this help file is intended for use by Authorized Data-Basics Support Personnel only. Incorrect implementation of the information discussed here can create problems with software functionality and/or data.

The startme.exe files distributed with versions 6.1.358.025 and 6.2.378.beta_0610 (and later) support a new logging feature that enables our support representatives to monitor the time it takes to execute the various phases of the login process.  Monitoring could be implemented if, for example, logon’s seemed to be taking an unreasonably long time for one or more users at a client’s site.

To implement this option (with an appropriate version of the startme.exe file), you must include a new command line option. To do so, change the startme shortcut to read:

startme.exe –timeLogon


The results can be found in the SAMPRO folder in a file named "<SAMPROUserId>~Logon.txt" that will be automatically generated by the system.
