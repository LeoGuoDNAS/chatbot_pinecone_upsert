Project Manager Workbench Cost Codes Tab

Purpose:

This tab displays a summary view of the costs to date for each of the Cost Codes for this Job.

Usage:

See Project Manager Workbench for details on this viewer’s upper tabs.

Important Fields:

Cost Code Id – Cost Code Name - Hrs To Date – Hrs Projected – Hrs Budget – Hrs Variance – Actual Cost – Committed Cost – Total Cost – Cost To Complete – Projected Cost – Budget Cost – Cost Variance

Since this information is pulled from Job History, only those cost codes which have Job History will appear. For each Cost Code, the above fields are summarized. Double-clicking on a line will launch the Cost Code viewer.
