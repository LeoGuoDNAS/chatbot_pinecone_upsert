Start Scheduled Event Manager

Purpose:

Starts Escalation Manager on your system.

Usage:

This function is required for Work Order and Web Log Escalation. This function may be run as a service on a separate machine. If the scheduled event manager is being started from a script, the script must use the

-startFunctions option  (i.e. -startFunctions=sschdldevntmngr)

Contact your Data-Basics Support Representative for details.

When restarting the Scheduled Event Manager, Process events will be run before sending e-mails to alleviate a sporadic timing problem experience by some users.

Support Only: see Suppress This Important Message is Brought to you by SAM Pro message to eliminate this message from emailed escalation messages with this sys-ffv-global-settings option.
