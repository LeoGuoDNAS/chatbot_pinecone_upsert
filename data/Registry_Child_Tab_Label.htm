Registry to Set the Child Tab Label Font

Purpose:

This setting may be used to adjust the fonts used for the child tab labels in SAM Pro viewers.

Usage:

This was implemented for use with the Work Order (and other viewers with MANY tabs) for systems with lower resolution screens. By selecting a ‘skinnier’ font or lower point size, these tabs will now be available for users with lower-resolution screens. Previously, the only way of handling this was to suppress tabs or shorten their names via registry. This was not desireable since, if a new tab was added to SAM Pro, it would not appear unless the existing registry was altered.

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Setting Values are Uppercase-Lowercase Sensitive:
The Setting Value is case-sensitive: be sure that you enter the value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.

Important Fields:

Id: sys-font-TabControlChild

See the Standard Registry section in Registry Viewer for additional Registry Id formats to allow entries for specific Users, Branches, Security, or User Groups to override the main entry.

Setting Key: Facename Setting Value: aFaceName

where aFaceName is replaced by a valid font name (e.g., Arial, Tahoma). This font must be installed on the users’ computer or it will not be used. Hint: Arial is the ‘skinniest’ font that is available on the vast majority of PC’s.

Setting Key: Bold  Setting Value: Yes or No

Specify yes to use the bold version of the font, or no to use the normal version. Hint: the regular versions of variable width fonts such as Arial are ‘skinnier’ than the bold version.

Setting Key: Points:  Setting Value: aPointSize

Where aPointSize: specifies the size in points. Point sizes ranging from 8-10 are recommended.

Example:

Id: sys-font-TabControlChild

Setting Key: Facename  Setting Value: Arial

Setting Key: Bold  Setting Value: No

Setting Key: Points  Setting Value: 9
