Registry Start Recompute T&M Revenue

Purpose:

In releases prior to 9.1.611.049, running this function always forced the Job Billing Item in Job History to match the Job Cost Code's Job Billing Item; this was not ideal for most users.

Usage:

The following registry should only be enabled if the Job Billing Item in Job History should be forced to match the Job Cost Code's Job Billing Item when this function is run (USE).  This registry is designed to be used by USE to maintain the previous default functionality where when the Job Billing Item in Job History WAS forced to match the Job Cost Code's Billing Item when this function was run.

Important Fields:

Id: sys-start-scmptejchunblldrvne

Setting Key:  ResetJCHBillingItemFromCostCode  Setting Value: true (default false)

Warning: This registry should NOT be set for most installations as it will cause issues with non-USE KPIs.
