Registry Report Performance Enhancement

The following registries enable performance enhancements for all or specific user-defined reports such as the Service Invoice that are importing into your database. Note that this does NOT apply to system reports such as the Purchase Order.

Apply Performance Enhancements to All or Selected User-Defined Reports

Id: sys-behavior-reports  

Setting Key: ReportExtended   Setting Value: true   (default false)

This option allows a user to invoke the performance enhancements contained in this case for all reports.

The registry setting ReportExtended can also be added to any report registry to enable (true) or disable (false) the performance enhancements.

For example, to enable for the Service Invoice only:

Id: sys-report-serviceinvoice

Setting Key: ReportExtended   Setting Value: true   (default false)

This will enable the functionality for the Service Invoice (assuming that the sys-behavior-reports registry is not enabled or set to false).

Pre-Compiled User-Defined Report Definitions to Reduce Load Time

Id: sys-behavior-reports

Setting Key: UseCompiled Report    Setting Value: true

This registry saves pre-compile report definitions to reduce the load time of any report, including the service invoice report.
Note: This registry is important for performance improvement for every report.
