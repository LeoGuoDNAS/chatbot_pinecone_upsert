New Record Button

If you are in a record viewer, the Blank Record button clears all data-entry fields (that are not set to automatically default) so that you can begin to create a new record. When you have finished entering the new information, select the Add Record Button to add this record to your database.

This button is also on the side button bar of the standard list view so that you can display a blank record to begin data-entry.
