Registry MSDS Printing (msdspdfpath)

Purpose:

The system allows the MSDS sheet to be a pre-generated PDF file. The system will print the PDF form of the MSDS if it exists and the SAMPRO text MSDS (entered in on the MSDS tab of the Inventory Item viewer) if not.

Usage:

The name of the MSDS PDF file for a given inventory item is of the form invntryitm_id.PDF.

If the location of the PDF reader of the PDF file cannot be located, operation will continue as before.

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Setting Values are Uppercase-Lowercase Sensitive:
The Setting Value is case-sensitive: be sure that you enter the value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.

Related Registry Entry

Registry Entry Adobe Reader

Important Fields:

Id: Msdspdfpath

See the Standard Registry section in Registry Viewer for additional Registry Id formats to allow entries for specific Users, Branches, Security, or User Groups to override the main entry.

Specify Path to PDFs

Setting Key: path Setting Value: pathtopdfs

where pathtopdfs is replaced by full path name of the folder holding the MSDS Pdf’s.

Examples:

Setting Key: path Setting Value: c:\databasics\msdspdfs

Setting Key: path Setting Value: \\myserver\sampro\msdspdfs
