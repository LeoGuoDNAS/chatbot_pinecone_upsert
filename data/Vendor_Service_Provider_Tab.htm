Vendor Service Provider Tab

This informational tab displays the Other Rates from the Other Rate Table specified in the vendor's Service Provider Rate Schedule (on the Main tab). Note that only ‘current’ rates (where today’s date falls within the date range specified on the Rate Schedule). Note that Service Provider Rate Schedules should ONLY include Other Rate Tables.

Id Description
The Id and Description from the Other Rate record.

Unit Cost Rate Type Start Date
These fields default from the Other Rate Table referenced in the Service Provider’s Rate Schedule.
