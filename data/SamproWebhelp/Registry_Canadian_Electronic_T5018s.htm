Registry Canadian Electronic T5018s

This registry is required for the Report AP Canadian Electronic T5018 function which creates the file that is used to submit the T5018s electronically to the CRA (Canadian Revenue Agency).

When Intercompany is enabled, the Canadian 5018EF report uses the Company record / Company Registry tab to specify the Company Name and Address and other registry settings required for these reports. This will reduce the need for multiple registries entries by Company Security. The new Company Record overload logic supersedes Registry values when/if present.

Company Name, Address: Use the Company Name & Company Address from the Company Record Header info if available.

Company Registry Tab Fields: Fields not defined in the Company Record header may be added to the Company record's Registry Tab.  For example,  T5018AreaCode and related fields for the 5018 electronic report.

First look for the Company Record's Registry entry. If not found,
Second look for the sys-report-rT5018E . If the registry is blank or not found,
Third look for the company specific companyname registry setting (e.g., companyname-1) and use that.  
If none of the above are available, the string 'companyname' will be used in the report instead of leaving it blank.

Registry Id: sys-report-rT5018E  

Registry Name: Report AP Canadian Electronic T5018s

The following settings define the contact information to be included in the electronic file.

Setting Key: T5018AreaCode  Setting Value: 3 digit code

Setting Key: T5018Telephone  Setting Value: nnn-nnnn

Setting Key: T5018Extension  Setting Value: up to 5 digit extension

The Extension registry may be omitted if it is not needed.

Setting Key: T5018ContactEmail   Setting Value: valid email address

Setting Key: T5018ContactPerson  Setting Value: Firstname Lastname

The following settings specify your company information which will be included in the electronic file.

Setting Key: T5018PayerAddress1 Setting Value: street address

Setting Key: T5018PayerAddress2  Setting Value: suite number

The Address2 setting may be omitted if it is not needed.

Setting Key: T5018PayerCity  Setting Value: city

Setting Key: T5018PayerPostalCode  Setting Value: postal code

Setting Key: T5018PayerProvince   Setting Value: 2 character province code

Setting Key: T5018TransmitterNumber  Setting Value: as specified by the CRA
