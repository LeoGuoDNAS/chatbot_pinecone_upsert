Report PR 401(k)

Purpose:

This report is designed to provide a printed and electronic report detailing 401(k) employee and matching contributions (if applicable) to be transferred to John Hancock or Pinnacle. This report must be run after you have computed and printed your payroll checks.

Registry:

See Registry PR 401(k) Reporting for the settings required for Pinnacle or John Hancock reporting.

Important Fields:

Start Date End Date
Defines the range of payroll check dates to be included on this report. To include only one day's checks on the report, the Start Date and End Date should be the same.

Branch Id:
If you need to limit the branches included on this report, enter the Branch Id (or double-click to select from a list).

Important Buttons:

 Click this button to run the report and create the electronic file for John Hancock or Pinnacle. If you entered a specific filename for the electronic version of this file in the Registry entry, the electronic version of the report will be saved as specified in the Registry.
