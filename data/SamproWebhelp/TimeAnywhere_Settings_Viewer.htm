TimeAnywhere Settings Viewer

Purpose:

This record is used by support to enter global configurations for TimeAnywhere.  It also records user configurations generated from TimeAnywhere itself.  It is recommended that these configurations be managed by support.

Additional Setup Notes:

Employee Viewer Setup - Other Tab

Remote Time Sheet Style (may be labeled TimeAnywhere Type):
Set this field to Individual if this employee will only be entering time for him or herself.

For Crew Managers, this field MUST be set to Foreman in order for them to see and log time for their Crews.

User Record Setup

In order to log onto TimeAnywhere, the Employee Id must be completed in the User record.
