Remote Access Query: wo_closing_script_headings

Purpose:

Defines the headings displayed in the Closing Call Script table in TechAnywhere.

Usage:

The contents of the closing call script table must be defined by the Remote Access Query: wo_closing_script_list.

Important Fields:

Application: TechAnywhere

Dataset Name: wo_closing_script_headings

Query Id: This field is assigned by the system when the record is saved.

Query Format: Frame

DataSet Query:

["Work Order Id",

"Question",

"Response",

"Ln"

]
