Start GL History Summary Rebuild

Purpose:

Start GL History Summary Rebuild is a starter-type function that completely reconstructs the contents of the GL History Summary.  The GL History Summary table improves performance for the DB Analytics KPIs that need financial data currently stored in the GL History (glhstry) table.  Essentially, this table contains a summarized version of the GL History.

Usage:

If DB Analytics is enabled on your system (see DB Analytics Overview ), a Schedule Event should be enabled to run Start GL History Summary Rebuild on a regular basis – ideally, at a light usage time when other backups are not running. This function has no input options which make it easy to schedule it for automated processing by the Scheduled Event Manager.
