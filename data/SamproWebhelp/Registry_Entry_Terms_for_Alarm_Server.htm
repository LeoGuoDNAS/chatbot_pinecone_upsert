Registry Alarm Server (terms)

Purpose:

This entry is required to specify the terms used in E-mails received by the Alarm Server.

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Usage:

If you are running the Alarm Server to receive external alarms, the following registry entry must be created. Note that if you are using this feature, the Client Site Equipment Id must be unique over all sites.
