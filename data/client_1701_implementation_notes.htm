The DayNite custom view opens up access to the fields necessary to manually record checks. This is only in the Day Nite customization. This functionality also depends on a registry entry to automatically assign a record Id value.

Id: sys-viewer-vchcklst
Contents: IdFromRn=true

To avoid confusion during the manual entry process also implementi the following registry option

Additional Contents: NewRecordAfterAdd=true

Without this setting the only way the user can initiate a new check record is to hit the New Record button.
