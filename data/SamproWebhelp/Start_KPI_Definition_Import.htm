Start KPI Definition Import

Purpose:

This function is used to import the KPIs (stored as .obj files) into your database. These files should be located in a subdirectory named KPIs located within the directory containing your SAM Pro executable.

Usage:

Make sure that the KPI’s that you important are the correct, up-to-date ones provided by Data-Basics. This function will update existing KPI’s within your database. The Start KPI Definition Export function may be used to export existing KPI’s before importing new ones.

Important Fields:

Range Screen:

Directory:
By default, this field will be set to a directory named KPIs located within the directory containing your SAM Pro executable. If your files are located in a different location it should be specified before clicking the Go button.

Checklist Screen:

Check the Do box to select the KPI’s you wish to import. Click the Go button to import the selected KPI definitions.
