New Call Required Resources

If the WO Materials Requisitioning module is implemented in your system (and the sys-ffv-global-settings registry’s UseWORequiredResources=true option is set), the Required Resources button will enabled on the New Call viewer’s side button bar. This button enables you to add required resources (inventory items and flat rates) to the work order you are creating AFTER selecting the Site, Equipment and other fields but BEFORE clicking New Call’s Add button.

When the Required Resource button is toggled to On, the lower section of the New Call screen change to display the Flat Rate and Inventory tabs.

FlatRate Tab

This tab enables you to add one or more Flat Rate categories to the Work Order you are creating. Materials and other required resources will be written to the Required Resource tab. A billing line will be written to the Other tab.

Quantity:
Enter the quantity of this flat rate category to be billed on this work order (typically, 1).

Flat Rate Id:
Enter a valid Flat Rate Id or double-click on this field to select from a list of Flat Rates.

Level (1 numeric)
Enter the flat rate pricing level (1-9) to be used for setting the default prices. Note that not all flat rate pricing services support 9 levels of pricing. Level 1 will be used if you do not enter a level.

Unit Price: 
This field defaults from the Flat Rate Category’s Pretax Price for the selected Level.

Billing Description:
This field defaults from the Flat Rate Category’s Description but may be adjusted by the user.

Description:
This display-only field defaults from the Flat Rate Category’s Description.

Inventory Tab

This tab enables you to add one or more required Inventory items to the Work Order you are creating. These items will be written to the Required Resource tab.

Quantity:
Enter the quantity of the item to be added to the Required Resource tab.

Inventory Item Id:
Enter a valid Inventory Item Id or double-click on this field to select from a list of items.

Cost Category:
Enter a material type Job Cost Category.
