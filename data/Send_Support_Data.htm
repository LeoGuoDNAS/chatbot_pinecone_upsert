Send Support Data Help Menu Option

This new option added to the Help menu is labeled 'Send Support Data'.  There is no group or user security provided so any user is eligible to execute the new option when instructed to by Data-Basics support. When this new option is selected:

A Dictionary of useful information from the user's current session is compiled. 

The user is prompted for a File Save location once the Support Data Dictionary has been compiled. 

Due to the potentially large size of the resulting object file, no automated link to the user's email has been provided. The user may choose to upload or email this file to support.
