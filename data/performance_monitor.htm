Performance Monitor

Purpose:

A Performance Monitor has been added to the SAM Pro Help menu so that it can be initiated in user mode. It provides statistics on how long takes to complete various operations, down to a fairly granular level. This may in turn be used by programming to evaluate areas to be targeted for performance enhancement.

Usage:

The Performance Monitor has been added to the SAM Pro Help menu. It is currently implemented

- On the Dispatch Board

- On the menu-type functions
If you turn the monitoring on in one of the menu-type functions (Viewers, Starters, Reports, Desktop, Folders), any function launched from that menu will have the monitoring turned on automatically.  This provides the option to evaluate performance during a function's launch process.

The Performance Monitor menu entry has three options: Capture Statistics, View Statistics and Clear Statistics.

Turn on Performance Monitoring: Capture Statistics

To turn on performance monitoring and Capture Statistics for a Dispatch Board operation (such as changing the select Work Order to Travel), you would select

Help > Performance Monitor> Capture Statistics (note that Capture Statistics is checked when enabled).

Next, click the Travel button.

View Statistics

To view the statistics for this operation:

Help > Performance Monitor> View Statistics

When you select this option, the system will open a workspace window with internal performance information on the operation just completed. This information is designed to enable programming to better identify areas where improvements in performance are possible.

Help > Performance Monitor> Clear Statistics

Clear Statistics

This menu option clears the statisitics for the Performance Monitor.

Turn off Performance Monitoring

To Turn Off the Performance Monitor, select

Help > Performance Monitor> Capture Statistics (selecting Capture Statistics when it is checked toggles it off).
