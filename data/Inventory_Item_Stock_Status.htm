Inventory Item Stock Status

Purpose:

This function runs a query that displays stocking details at the primary location that you specify, as well as availability as other locations.

Even if multinational currency is implemented on your system, this report will show all values as converted to your firm’s local currency.

Important Fields: Initial Range Screen

Primary Location – Location Id:
Enter the Id of the Location that you wish to check the status on. This field is required.

Inventory Item Id:
Inventory Item Name:
To limit the results of this query to specific Items, enter a list or range of Inventory Id’s or Names.

Inventory Group Id:
To limit the results of this query to a one or more Inventory Group or Groups, enter a list or range of Inventory Group Id’s.

User Id:
Enter this range to limit the items to those assigned to this user specified on the Inventory Item’s Main tab.

Vendor Id:
Enter this range to limit the items to the primary Vendor specified on the Inventory Item’s Purchasing tab.

Click the Go button to display the Stock Status at the Location you selected.

Important Fields: List Screen

Inventory Item Id: Inventory Item Name:
 The Inventory Item Id and Name.

Location Id:
The primary Location that you selected in the initial range screen.

PriOnHand PriAvail  
The On Hand and Available quantities of this Item at the primary location.

Unit Cost:
The Cost displayed is based on the Item's Cost Method (average, last or standard).

OthOnHand: OthAvail:
The Item’s On Hand and Available quantities at all locations other than the primary.
