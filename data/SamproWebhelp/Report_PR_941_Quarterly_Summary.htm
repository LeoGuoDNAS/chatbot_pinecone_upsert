Report PR 941 Quarterly Summary (RPR941S.rpt)

The Report PR 941 Quarterly Summary (RPR941S.rpt) provides a better input document for the Federal form. This topic describes the report PRIOR TO the Covid-19 additions.  See Report PR 941S with Covid-19 Support for the updated report.

Purpose:

This report is designed to assist the user in completing the Employer’s Quarterly Federal Tax Return.

Usage:

This report is generated with Payroll History data as well as user values entered on the Report Range screen.

Important Fields: Report Range Screen

PR Year: PR Quarter:
The year and quarter you wish to run this report for.

(5b) Social Security Tips
SAM Pro does not track Social Security Tips. This data entry field may be completed by the user if Social Security Tips should be reported.

(7b) FICA Sick Pay Adjustment.
(7c) FICA Other Pay Adjustment
Adjustments (not reflected in PR History) to FICA Sick / Other Pay for this quarter should be entered in these fields.

(9) Advanced EIC Payments
SAM Pro does not track this information. These data fields should be completed by the user if advanced earned income credit payments have been made to employees this quarter.

(6a) Number of Qualified Employees First Paid Exempt Wages this Quarter
(6b) Number of Qualified Employees Paid Exempt Wages this Quarter
(6c) Exempt wages/tips paid to qualified employees this Quarter
SAM Pro does not track this information. These data fields should be completed by the user if exempt wages were paid to qualified employees this quarter / first paid this quarter.

(11) Total Deposits, including Prior Quarter Overpayments
In the earlier report, the report range screen provided input fields for Prior Quarters Federal Tax Adjustment and Total Deposits for Quarter. In the updated report, these fields have been lumped together in this input field.

(12a) COBRA Premium Assistance Payments:
(12b) Number of Individuals Provided COBRA:
SAM Pro does not track this information. These data fields should be completed by the user if COBRA assistance was provided this quarter.

Description:
Enter any text that you wish to appear on the report header in this field.
