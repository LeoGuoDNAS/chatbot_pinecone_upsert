Data Entry Keys

Moving the Focus From Field to Field in a Record

Tab,Enter and Shift+RightArrow move to next field (forwards).
Shift+Tab and Ctrl+G moves to the previous field (backwards)
Shift+LeftArrow moves to the previous field (backwards).
Shift+UpArrow moves to the first field.
Shift+DownArrow moves to the last field.
UpArrow moves to the previous adjacent field (from line 2 to line 1 in an address).
DownArrow moves to the next adjacent field (from line 1 to line 2 in an address).

Moving the Focus From Line to Line in a Data-Entry Child Tab

Alt+DownArrow to move the focus to the same field in the next line.

Alt+UpArrow to move the focus to the same field on the previous line.

Copy Field from Previous Row

Ctrl+F (in a child table data entry field) copies a field's value from the line above. This functionality is not available for button or boolean fields.

Tip: Setting the Date Field in Multiple Rows
1. Set the date in the first line to the desired value but do not tab out of the field.
2. Alt+DownArrow to move to the same field in the next line.
3. Ctrl+F to set the field to the same value as the field in the previous line.
4. Continue with Alt+DownArrow  and then Ctrl+F to set all the lines.

Copy and Paste

Ctrl+C copies the contents of the current field (from the caret to the end of the field) to the Windows clipboard.
Ctrl+V Pastes the contents of the Windows' clipboard to the field with focus. The system removes the pipe symbol before pasting text into SAM Pro fields.
Right-Click in Non-Id Fields: When the user right clicks on a non-Id, data entry field a menu pops up. This menu displays the following options:

Copy (Ctrl+C)
Paste (Ctrl+V)
Customize Field (if allowed for the User's Group)

To select an option, use the Up or Down arrow key to highlight the option, and then press Enter to select it. To exit the menu without a making a selection using the keyboard, press Esc.  

Right-Click in Notes Text: When the user right clicks in the Notes text field, a menu pops up with the options of Copy and Paste.  To select an option with the keyboard, use the Up or Down arrow key to highlight the option, and then press Enter to select it. To exit the menu without making a selection using the keyboard, press Esc (Escape).

Notes text may be selected to copy by holding down the left mouse button while scrolling. The selected text may be copied via the right-click menu, or by pressing Ctrl+C.

Right-Click in Id Fields: Normally, right clicking on an Id field will bring up a Viewer for the parent record. If the user would like to see the above menu instead, the user should hold down the Control key while doing the right click.

Copy Field from Previous Row

Ctrl+F (in a child table data entry field) copies a field's value from the line above. This functionality is not available for button or boolean fields.

Caret Movement Within a Field

LeftArrow, RightArrow moves the caret one character position.

End moves the caret to one character past the last position

Ctrl+End moves the caret to one character past the last position in a multi-line text box

Editing in a Left-Justified Field

Backspace erases the character to the left of the caret and shifts the remaining characters left.
Delete erases the character at the caret and moves the remaining characters to the left.
Home moves the caret to the leftmost position in the field.
Shift+Del erases from the caret to the end of the line.
Ctrl+Del erases entire contents of the field.
Shift+Home restores the original contents of field.

Insert VS Overstrike in a Left-Justified Text Field

If the field already contains text, and you tab or click into it so the caret is at the beginning of the field:

If you immediately begin typing, the new text you enter will overwrite the existing text

If you press the insert key before entering text, the text you enter will be inserted at the beginning of the line in this field. When you tab or mouse out of the field, the insert behavior will be as above.

If the field already contains text, and you tab/click into and then move the caret (with the arrow keys or mouse), the new text you enter will be inserted at the current caret position

Editing in a Right-Justified Field

Backspace erases the character at the caret and shifts the remaining characters right.
Delete erases the character at the caret and moves the remaining characters to the left.
Home moves the caret to the right-most position in the field.
Shift+Del erases from the caret to the beginning of the line. It sets a Date field to 01011900 (the 'blank' date).
Ctrl+Del erases the entire contents of a text or numeric field.
Shift+Home restores the original contents of the field.

Date Fields

Spacebar enters the current date in a date field.
+ or - increases/decreases the date in a date field.
Shift+Del sets a Date field to 01011900 (the 'blank' date).
The year defaults to the current year when you begin to type in a blank date field.

Editing in a Decimal Field

If the caret is positioned to the right of the decimal point, follow the rules for editing a left-justified field. If the caret is positioned to the left of the decimal point, follow the rules for editing a right-justified field.

Focus Movement Within a List

UpArrow moves up one line.
DownArrow moves down one line, and also positions the focus within the list.
PageUp and PageDown move the focus 10 lines up or down the list.

Other Key Functions

Spacebar selects (pushes) a button when the focus is on it; is also used to select/deselect radio buttons and flag fields and to enter the current time in a time field, and the current date in a date field.
F2 or Alt+F7 pops up a list of values for the Id field with focus (equivalent to double-clicking on a field).
Alt+F7 pops up the Clock viewer for the Time field with focus, or the Calendar for the Date field with focus.

Ctrl+Z zooms so that the selected pane files the entire window; pressing Ctrl+Z again toggles back to the regular display. This functionality may also be accessed by selecting by selecting the View>>Zoom from the menu bar. Note that user is not permitted to access Single View (with the Magnify button) while in Zoomed mode.

Ctrl+Q: Toggles between the Magnify and Line views of the selected child tab line.

Hotkeys

Special Hotkeys enable the user to push buttons from the keyboard without using the mouse. Click on Side Bar Buttons for details!
