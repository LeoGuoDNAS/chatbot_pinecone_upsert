Work Order Tech Status Tab

This tab will only be displayed if the ShowWOStatusList=true option is set in the sys-viewer-vwrkordr registry entry.

This tab is designed to provide simple access to information about a tech's status on a WO.  The information about each tech found on the WO Tech tab is summarized into a single line on the tab. This feature is suggested for use in conjunction with TechAnywhere, as it reflects the status that is currently listed on the technician’s remote device.

The Start Work Order Status Update function updates this tab for existing work orders.
