Project Manager Workbench Quotes Tab

Purpose:

This new tab displays the Quotes associated with the selected Job. Doubling-clicking on a line will open the selected Quote.

A Quote will appear on this tab if the Job Id is completed in the Quote header. A Quote will also be displayed if its Job Id is blank but the Work Order Id is completed and references the selected Job. The following fields are displayed:

Quote Id - Description – DateOpened – Status

These fields default from the Quote header.

Equipment Id – Option – Bill Item Id – Bill Item Status – Price – Scope

These fields default from the Quote Scope tab. Note that the Bill Item Id and the Bill Item Status are only completed if a Change Order Billing Item has been generated for the specified option.
