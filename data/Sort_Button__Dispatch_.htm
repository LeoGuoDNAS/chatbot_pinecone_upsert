The Sort button enables you to select the order of the Work Order List. When you push the Sort button, a list of sort orders will appear. Click on the desired sort order. The Date/Time/Technician option is especially useful at the beginning of a day, since it lets the dispatcher see at a glance, which WorkOrders are up first in technician order.

The Technician/Date/Time and Date/Technician/Time options are recommended for use with the Graphics view of the Work Order list.

The Status/WorkOrderId option allows all Work Orders of a specific status to be grouped together and in Work Order Id order.
