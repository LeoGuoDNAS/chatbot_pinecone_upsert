Quote Reason

Purpose:

Defines the reason for which a Quote is being generated and/or declined. Alternately, some firms may wish to use this record to define the source of the quotation – direct mail, radio ad, etc.

Usage:

Referenced in the Quote Master Viewer and CSRAnywhere.

Important Fields:

Reason Id:
Uniquely defines this record.

Reason Description:
Briefly describes the reason for the Quote. In the Quote Master viewer, this Description will default into the Quote Scope of Work tab when you select the Reason record.

Security Id:
If you leave this field blank, users of all security levels with be able to ‘see’ this record. If you wish to limit read access of this Reason to users of a specific region or security type, enter a valid Security Id.
