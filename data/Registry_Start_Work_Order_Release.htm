Registry Start Work Order Release (sys-start-srlsewrkordrs)

Related Registry:

Registry Shop Order (required)

Important Fields

This registry setting allows specification of the Work Order Log Type for WO-Tech lines that get released as a part of the Material Requisitioning process. The new setting, when present, overrides the default functionality (the Log Status on the Tech tab line is set to the default value for that kind of tech or via registry (sys-web-access)).

Id: sys-start-srlsewrkordrs

See the Standard Registry section in Registry Viewer for additional Registry Id formats to allow entries for specific Users, Branches, Security, or User Groups to override the main entry.

Setting Key: ReleasedStatus  Setting Value: work order log type id
