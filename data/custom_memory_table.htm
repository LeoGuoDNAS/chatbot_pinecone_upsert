Custom Memory Table

Purpose:

A new table has been added that enables support to restrict the contents of a SAM Pro-maintained list (such as the WO Log Type statuses) to specific values.  Records are maintained with the View Custom Memory Table function. 

If you are logged on as the super user, you will see the standard SAMPRO list instead of the custom lists implemented with this function.

NOTE:  you may need to run the Start Function List Update process in order to see the new viewer function.

Usage:

Great care must be taken in implementing this feature. If a Custom Memory Table is implemented for a specific list, you MUST create a Customer Memory Table record for each value that you wish to appear in the list.

Be sure that all values that are required for system functionality are included as Custom Memory Table records. For example, if you are implementing records for the work order log type, you must include records with specific system meanings such as waiting service, invoice ready, unacknowledged, etc.

One viewer that has yet to be addressed is the Dispatch Board.  Currently, this implementation would not prevent a user from setting an invalid WO Tech Status by pushing one of the existing sidebar buttons whose associated status is not in the customized values list.  Please advise of any other issues that I'm forgetting.
In other viewers, both the double-click and field validation code is aware of the custom Id values.

Important Fields: Upper Section

Table Name
This field is restricted to valid SAMPRO memory table names.  Special double-click logic is implemented for this field but it also supports manual data entry.  Once a valid Table Name is selected, a list showing the table's standard SAMPRO values is displayed at the bottom of the entry form.

Id
The Id field is restricted to valid Id values for the selected Table Name displayed on the lower section of screen. The Id must be entered EXACTLY as it appears on the Standard List tab; the user is NOT permitted to invent their own Ids.

Name
A Name is required for each record but the values are NOT restricted to standard SAMPRO values. The Name will appear in the alpha fields of pick lists.

Important Fields: Standard List Tab

Id Name

This list displays SAM Pro’s standard values for the selected table. The Id that you enter in the upper section of the screen must match one of the Id’s exactly.
