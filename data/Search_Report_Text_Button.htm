Search Report Text Button

This button is located on the side button bar of the Data-Basics Report viewer to enable you to quickly locate a word or phrase in the report. When you click the Search Report button, the Find box will appear.

Find
Enter the text that you wish to located in the Find text box. Note that the system will look for an exact match.

Case sensitive
By default, Case Sensitive is enabled: the system will look for an exact match (including upper and lower case) when you click the Find button. If you want Search to locate a string of alphanumeric characters ignoring upper- and lowercase, blank the Case Sensitive box by double-clicking on it.

Direction
If you are at the beginning of the report, select Forward (the default) to search the report from beginning to end. If you are at the end of the report, select Backward.

Click the Find button to locate the text you have entered. If the system locates a match, it will highlight the text in green, and position the Find box on that page. Click the Find button to locate the next instance of the text, or Cancel to end the search. Note that all occurrences will be highlighted in green.
