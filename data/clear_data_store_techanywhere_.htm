Clear Data Store (TechAnywhere 2.0+)

Purpose:

This TechAnywhere function enables you to clear the local datastore on a TechAnywhere Tech’s wireless device. This would be run if a tablet were to be reassigned from one technician to another. In this way, the TechAnywhere software is retained on the tablet.

Usage:

This function is implemented by running the file clearDatastore.hta. This file is located in the techanywhere directory on the device. We do not recommend that you create a desktop icon to point to this file as it should only be run in rare circumstances.

1. Have tech use Window’s Explorer to browse to C:\techanywhere and then double click on clearDatastore.hta.

2. The user will be asked to confirm the operation by clicking OK or to end the operation by click Cancel.

3. The operation will complete within 5 seconds.

4. The user will be asked to close the application by clicking OK.
