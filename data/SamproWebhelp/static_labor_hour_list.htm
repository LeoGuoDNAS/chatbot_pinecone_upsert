Remote Access Query: static_labor_hour_list

Purpose:

Specifies the hours worked that will appear on the drop-down list that technician may select from when adding time to a work order.

Application: TechAnywhere

Dataset Name: static_labor_hour_list

Query Format: Frame

Dataset Query:

Lists the hours that will appear on Add Edit Labor page in the Hours drop down box. These values are entered as literals using the format "n.n" or "n.nn" (half hour or quarter hour increments) using the frame format: ["n.n", "n.n", ...]

Example:

 [

"0.5",
"1.0",
"1.5",
"2.0",
"2.5",
"3.0",
"3.5",
"4.0",
"4.5",
"5.0",
"5.5",
"6.0",
"6.5",
"7.0",
"7.5",
"8.0"

]

Notes:

0.0 is not a desired value as it is too easy for a technician to add and submit it.
