Starter Viewer

Purpose:

Start a function / process.

Click this button at access a list of starter functions.

Example:

Starter functions enable you to take Compute Finance Charges, generate PM Work Orders, etc.

Usage:

Enables the user to quickly identify and start processes. All of the starter functions available to your user group are listed in the Starter window. To start a function, double-click on it with the left mouse button. Alternately, click on the starter function once to select it and then click the Go button.

Important Fields:

Click the Help button from the initial Starter Screen to display information on its usage and how to complete its fields.
