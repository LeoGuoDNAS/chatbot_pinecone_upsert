The Help button enables the user to access Data-Basics' context-specific Help topics and on-line manual.

1. If you press the Help button from a list view or take-back viewer, the system will display basic information on accessing or taking back a specific record.

2. If you press the Help button from within a record, context-specific Help should appear. If does not, use the Help window's Contents or Index button to locate the appropriate topic.
