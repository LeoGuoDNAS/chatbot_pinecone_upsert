Report PR W-2 Box 14 Covid Addendum

The IRS notes additional reporting requirements for the W-2 in its “Guidance on Reporting Qualified Sick Leave Wages and Qualified Family Leave Wages Paid Pursuant to the Families First Coronavirus Response Act.”  This specifies that employers either provide a statement with the W-2 specifying COVID related wages for 2020 or print the information in W-2 box 14 (not feasible due to size issues on the W-2 for a number of reasons).  

Data-Basics’ Report PR W-2 Box 14 Covid Addendum should be installed and run to support this requirement if any wages were paid applicable to the Families First Coronavirus Response Act.  This report uses the same information as the revised 941 Summary report. See Report PR 941S with Covid 19 Support for details.

The SAMPro Report PR W-2 Box 14 Covid Addendum (RPRCOVID.rpt dated 2021-01-26 or later) supports SAMPro versions 8.2 and above. A version for pre 8.2 users is also available (RPRCOVID [pre-82].rpt revised 2020-01-26 or later).
