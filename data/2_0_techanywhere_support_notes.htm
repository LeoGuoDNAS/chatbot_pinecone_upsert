1701 Submit Backup to Databasics Button

The "submit log file to databasics" button has been replaced with a "submit techanywhere folder to databasics", on the sync page.

When the user clicks this button, a backup is created and zipped, then transferred via ftp using config entries: configFTPHost, configFTPUsername, and configFTPPassword. The zip filename includes the config entry configDBClientNumber.

The tech is asked to enter a problem description.  This is put in an email and sent to dnapoleon@databasics.com along with the zip filename that was uploaded to the databasics ftp server.  The company logo image is sent to to help identify the client.

The email is sent using smtpServerForLogSubmission,smtpUsernameForLogSubmission, and smtpPasswordForLogSubmission.
