1.0_20080808 / vw_7.6_20080808

- Added metrics RAQ to the visualworks image.

- Metrics links are now colored red if they have unanswered metrics.

- You cannot complete a work order if there are unanswered metrics in the current equipment list.

1.0_20080807 / vw_7.6_20080807

- Fixed a bug in the charges grid that displayed an error message when tabbing out of the unit price. 

- Modified the way closing call script responses get retrieved.  Now I iterate over the call scripts and check for questions that are the same as the closing script questions.  If they are the same I take the response and the serverRN and put it in the corresponding closing script line.  I then remove the call script line.  When the closing scripts are submitted the server will then use the serverRN to update the call script in the work order.
