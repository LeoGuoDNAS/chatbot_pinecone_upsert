Internal Registry TechAnywhere Static Files (sys-start-stchanywhrecrtesttctxtfle)

Purpose:

The function Start TechAnywhere Create Static Text Files is used to generate the static files (such as parts lists) required by TechAnywhere. This registry specifies the location in which these files will reside.

Important Fields:

Id:

 sys-start-stchanywhrecrtesttctxtfle

Contents/Additional Contents

 StaticFileDirectory=path

path should be replaced with the path to the folder on your TechAnywhere servier in which these files will reside. This folder must be accessible by the TechAnywhere synchronization process.
