Job Templates Viewer

Purpose:

This new viewer is designed for the creation and maintenance of the template records used in job cloning. Since this is a new viewer, it must be assigned to a user group or groups in order for it to be accessed. We strongly recommend that access be restricted to only those user groups that really need to be able to create or maintain template files.

See Job Template Overview for an overview of the Template files.

Usage:

Once the Job Template has been created in the special Job Templates viewer, it can be used in regular Job (and New Call) viewers for Cloning. A Template may be selected in the Job viewer for Cloning â€“ but the selected Template cannot be changed in the Job viewer.

Jobs should NOT be cloned from the Job Template viewer.

Important Fields:

Only those fields that are common to all Job records that be created from this template should be completed. Please refer to the Job Viewer for details on setting the other fields.

Template
This field must be checked in order to flag this as a template record created for job cloning purposes. Job template records may only be created and modified in this viewer.

Client Site Id:
If you are using New Call 2 - COD processing with this Job, the Client Site should be blank. If this field is completed (for regular template jobs), the system will validate that the Client Site Id points to a Client Site Template.
