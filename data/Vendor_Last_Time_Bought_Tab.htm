Vendor Last Time Bought Tab

Important Fields:

The new Last Time Bought tab displays a list of the most recent purchases of all inventory items bought from this Vendor. You may view the Purchase Order detail for a line by double-clicking on it. The following information is displayed for each inventory item:

Item No. Description  PO Number Quantity Unit of Measure

Unit Cost Entry Date Location %OT Prmd %OT Rstd

The % OT fields are reserved for a future release.
