Registry Dispatch Classic/Day Board Status Colors (sys-dispatchboard-colors-status)

This registry entry controls background color of the default Classic and Day Board cells based on the technician status (e.g., onsite, waiting service, travel, etc.) of the work order.

Usage:

Statuses that are not specified in this entry will retain the default background color of the cell.

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly. You will need to close and reopen the Dispatch Board to see this registry take effect.

Important Fields:

Id:

 sys-dispatchboard-colors-status

Specify Status Colors

 Setting Key: statusId Setting Value: color:Custom

where statusId is replaced by a log type id and color is replaced by the Windows color map index.

Important Note: all registry options that use the color functionality must be correctly configured or a system primitive error will appear when the Dispatch Board is opened. These entries will NOT be ignored if an ‘x’ is placed in from of them to temporarily disable them.

A. Verify that the registry has been saved.

B. To enter the index number, enter all information up to the word Custom. For example, to set the status with a log type id of travel, you would type travel=color:

C. After entering the colon: double-click on the line to display the Windows color selector palette, click on the desired color and then click OK.
