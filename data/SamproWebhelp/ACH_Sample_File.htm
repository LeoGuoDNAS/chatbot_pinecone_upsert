ACH Sample File

The following shows sample ACH output structure. Please note that individual results will vary depending on your entries in Registry ACH/Direct Deposit Entry as well as the individual data being sent.

The first characters of each line (record) indicate its purpose:

File Header  1,01  Record

 Batch Header 5,200  Record (note: we only support 1 batch per file)

 Batch Detail 6 Records (characters following the 6 indicate transaction type)

Batch Control 8,200 Record

File Control  9 Records
