Registry Start Project WO Generator (sys-start-sprjctwrkordrgnrtr)

Purpose:

Control handling of closed Jobs.

Related Registries:

This function is also sensitive to the required fields registry settings for sys-start-snwcll. If a field is required in New Call it must be completed or defaulted when this function is run. If not, an information box describing the issue will be displayed and the Work Order will not be generated.

This Registry may be used to specify which lines from the Site WO Other Tab should be added based is the Work Order's Service Category.

Important Fields:

Project WO Generator Registry to Err / Warn on Closed Jobs

Start Project Work Order Generator has been refined so that when closed jobs are pulled into the checklist, the Do button will not be checked. A new registry (below) will control if you are allowed to change the Do flag.

Note: the Job's Date Closed is compared with the functions Date Must Complete.  A Job is considered 'closed' for this function if it's Date Closed precedes the Date Must Complete.

Registry Id: sys-start-sprjctwrkordrgnrtr

Setting Key: ClosedJob      Setting Value: error  (default warning)

In absence of registry or with setting value "warning" allow user to check Do, and give warning. If registry is set to error, do not allow user to check Do, and display an error message.
