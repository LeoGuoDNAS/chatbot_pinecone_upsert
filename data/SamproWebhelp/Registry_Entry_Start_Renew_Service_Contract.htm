Registry Start Renew Service Contract (sys-start-srnwsrvcecntrct)

The Start Renew Service Contract function advances the Job Date Closed (if set) by one year. This function was also advancing the Job Date Open field by one year, which most clients feel is undesirable. With this release, only the Date Closed (if initially set) is adjusted by default.

The following registry setting enables users to re-implement advancement of the Job Date Opened when Renew Service Contract is run.

Id: sys-start-srnwsrvcecntrct

See the Standard Registry section in Registry Viewer for additional Registry Id formats to allow entries for specific Users, Branches, Security, or User Groups to override the main entry.

Setting Key: AdvanceJobDateOpened Setting Value: true

If this option is specified, then the Job date opened will be advanced 1 year. The default is not to advance the job date opened.
