Equipment Type Viewer

Purpose:

Organizes the equipment that your company installs and services into types.

Example:

You could create equipment types for boilers, A/C's, furnaces, etc.

Usage:

Referenced in the Client Site Equipment, Equipment Master, and Inventory Item viewers.

Master Contract/PM Master Schedule
This record is also referenced in the Master Contract (PM Master Scheduler) Provider Rate tab. It enables the system to ascertain which Service Provider costs / work order revenues should be associated with a particular PM when a PO is automatically generated to the vendor specified in the Technician record. See for PM Master Service Provider Rates for details.

Important Fields:

Equipment Type Id:
A unique Id for this Equipment Type. This field is required.

Equipment Type Name:
A brief description; this field is optional.

Obsolete?
If this field is checked, the record is considered Obsolete, and it may not be directly specified during data-entry (either by entering the Id or selecting it from a list). Obsoleted records can be edited and the Obsolete option can be un-done when necessary. When you check the Obsolete field, the following message box will be displayed:

“You have elected to obsolete this record. Would you like to see existing references to this record?”

If you click Yes, the system will display a viewer that provides click-through access to all tables referencing this record: see References List for details.
