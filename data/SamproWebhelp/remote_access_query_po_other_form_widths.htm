Remote Access Query: po_other_form_widths

Purpose:

Controls the widths of the cells in the Purchase Order Misc. Form on the Add/Edit Purchase Order page.

Usage:

By default, the elements in the PO Misc (Other) Form are set to the following widths in pixels: [100,100,100,150,60,50]. This RAQ enables you to change these specifications or to allow the system to automatically accommodate the character widths.

Important Fields:

Application: TechAnywhere

Dataset Name: po_other_form_widths

Query Id: This field is assigned by the system when the record is saved.

Query Format: Frame

DataSet Query:

To automatically accommodate the character widths of the table (open bracket close bracket, no space in-between):

 []

To specify the width of the Quantity, Unit Cost, Other Rate Id, Description, Equipment Id and PO# columns:

 [pixels1,pixels2,pixels3,pixels4,pixels5,pixels6]

 where pixels1-6 are the replaced by the width in pixels.

  Example: [100,100,100,150,50,50]
