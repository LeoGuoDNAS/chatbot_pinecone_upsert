Registry Technician Viewer (sys-viewer-vtchncn)

Purpose:

This registry entry should be implemented if you wish to enable the printing of a custom technician evaluation report from the technician viewer.

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Related Registry:

Set WO Status by Tech Web Access (sys-web-access)

Important Fields:

Id:  sys-viewer-vtchncn

See the Standard Registry section in Registry Viewer for additional Registry Id formats to allow entries for specific Users, Branches, Security, or User Groups to override the main entry.

Specify the Report for the Print Evaluation Report Button

Setting Key: PrintTechEvaluationReport Setting Value: functionid

Where functionid is replaced by the User-Defined Function referencing the report you wish to appear.  Report Service Provider Evaluation (with the Function Id RSRVCEPRVDR) is recommended. You may need to imported this report into your database.
