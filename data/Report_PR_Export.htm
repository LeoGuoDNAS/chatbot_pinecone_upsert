Report PR Export

Purpose:

The function Report PR Export creates a text file that payroll companies will use to generate paychecks. A report for the user to review/file will also be printed.

Paycor, Paychex, PrimePay, ADP, and GMS versions are currently supported; the report version that is generated is controlled via registry.

Usage:

This functionality requires that a vrgstry.cus file with the necessary permissions be in place. It may be necessary to run Start Update Functions to see this report. Note that this report is part of the software itself and does not have to be imported into the database.
