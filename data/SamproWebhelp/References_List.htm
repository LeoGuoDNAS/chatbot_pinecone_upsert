References List

Purpose:

This viewer allows you to access and maintain your data by accessing the records that reference (point to) the record you launch this viewer from.

Example:
Suppose you flag the Obsolete box in the Account Manager record for Bob Jones. The system will display a list of tables (records) that reference the Account Manager Bob Jones so that you can change to a different Account Manager. The Row Count Column lists the total number of records referencing this Account Manager. The Account Manager can be referenced by the Client, Client Order, Client Site, Client SiteEquipment, MasterContract, and WorkOrder records. Typically, you would want to change the Account Manager in your static records (Client, Site and Equipment).

Usage:

The References viewer is displayed when a user checks the Obsolete box, or select References from the Tools menu.

Important Fields:

The References viewer will display a list of tables (records) that reference the specific record that you invoked it from. This record will appear in the top border of the Viewer.

TableName ChildtableName DataBaseColumnName
The Viewer / Tab that references the record you lauched this viewer from.

Double-click on a line to display a filtered list of this type of records. The record id range will be set the References function so that only active records will appear.If, for some reason you needed to view inactive records pointing to your original record, click the Filter button and adjust any Closed/Inactive/Obsolete ranges as needed.
