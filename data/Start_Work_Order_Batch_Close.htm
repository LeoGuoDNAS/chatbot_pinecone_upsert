Start Work Order Batch Close

Purpose:

This function enables you to assign a Date Closed to your Work Orders. Normally, the Date Closed is set when you mark the work order Invoice Ready or Cancelled, and click the Change Record button, and click Yes to prompt asking if you wish to close the Work Order.

This function makes the automatic Filtering features more useful, since, by default, only open work orders will appear on the Work Order list. Note that this change does not affect the Dispatch Board.

Usage:

When this function is run,  the processing classifies and counts WO Tech lines as either Open (not Cancelled or Billed) or Closed (Cancelled or Billed).  If a WO has any Open lines, it is excluded from the list. Only Work Orders where all Tech lines are Cancelled or Billed will be eligible for closure unless modified via registry. If the status of all lines is Billed or Canceled, the Work Order will appear on a checklist. For each WO that you check, the system will enter a Closed Date equal to the largest date (if set) on the Technician tab (or to the Closed Date that you specify). An entry to the Work Order’s Change Log will be made when the Work Order is closed.

Registry:

See Registry to Add/Change Statuses in Start Work Order Batch Close to define which statuses are eligible for this function.

Important Fields: Initial Range Screen

Closed Date:
This field enables you to specify a default Date Closed for the Work Order if all the Date Scheduled fields on the Work Order’s Technician tab are blank.

Date Opened
Work Order Id
Client Site Id
Client Id
 By filling in one or more of these range fields, you can control which eligible work orders will be selected to be marked closed. We recommend that you complete at least the Date Opened field.

Examples:
To mark eligible work orders created in 2016 closed, you would fill in the Date Opened range field as follows:

2016-01-01_2016-12-31

To mark only those work orders for Client ACME that we created in 2015 closed, you would fill in the Date Opened and Client Id fields as follows:

 2015-01-01_2015-12-31
 ACME

 When you click the Go button, a list of work orders eligible for closure will appear.

Important Fields: List Screen

Do
This field enables you to select which work orders will be closed. By default, all work orders are selected. Double-click on the Do field to select or deselect an item (alternately, press the spacebar when the caret appears on this field); an "x" appears in the Do field when that work order is selected.

Date Closed:
By default, the Date Closed is set equal to the largest date on the Technician tab. To adjust the proposed date closed, click on this field, enter the desired date in mmddyyyy format, and then press Enter. Alternately, you may enter today's date by clicking the Spacebar.

Work Order Id: Client Site Id: Client Id: Work Order Name: Date Opened:
These fields are displayed for informational purposes so that you can verify which work orders should be closed.

Important Buttons:

Click the Check button to check (select) every item on the list.

Click to de-select every item on the list.

 When you click the Go button, the system enters the Closed Date specified on the list for all Work Orders that are checked.
