Client Site Equipment MasterSchedule Tab

By default, this tab lists upcoming inspections associated with the specified Master Contract for which a PM Work Order has not each been created.  The following fields are displayed for each inspection line:

Contract Id:  Inspect Date: Hours: Cost: Value:

However, with the UseMasterContractEquipmentTable global registry set to true, processing for the MasterSchedule list is produced as noted below using the Advanced Analytics function.

Contract Id:
Lines without a Contract Id were entered on the Client Site Equipment PM Schedule tab. Lines with a Contract Id were manufactured from a Master Contract Id.

Source:
When the Source is 'MCE', the line is predicted based on the Master Contract definition.

When the Source is 'CSE', the line exists on the Client Site Equipment PM Schedule tab.

Work Order:
When a 'CSE' line has a Work Order Id, it has already been through the Create PM Workorder process. 'CSE' Lines with a Contract Id If but no Work Order Id were manufactured through the Build Out Process but have not yet had the PM Work Order generated.
