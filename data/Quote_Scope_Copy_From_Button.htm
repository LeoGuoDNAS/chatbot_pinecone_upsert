Quote Scope Copy From Button

This button is located on each Quote Scope line in Quote Master and on the Quote Scope viewer's Main tab.  Pushing the button launches a wizard that lets the user complete the selected Scope by copying the details from other existing Scope for this Quote. Labor, Inventory, Material, Sub and Other lines associated with these Scopes will be imported.

Note that the user is responsible for managing revenue adjustments: these will NOT be generated if a line is copied.

To use this feature:
Your Quote must have one or more options that you wish to copy from. Next, add a new Scope line, complete its Option Code, change (or select) the Equipment Id if needed, and Save the Quote.

Select the Scope Option line you just added, and then click the Copy From button. The Copy From Options Wizard will appear:

Copy From Options Wizard – Select Option Codes Screen:

Ok – Option Code – Quote Amount – Fixed Price – Scope of Services

This screen allows you to select one or more existed options to update the selected scope option with – flag the OK box to select the Option Codes to Copy From. If more than one Option Code is selected, system will include details from all selected options and combine the Scope of Services text.

Click the Next button to continue.

Copy From Wizard Options Wizard – Edit Option Screen

Fixed Price:
If this is a Fixed Price quote, this field will default from the Option(s) you selected in the previous screen. It may be adjusted if needed.

Note: If you are cloning from a non-fixed fee quote and enter a fixed fee in the wizard, the prices on the detail lines for the new option will be cleared.

Quote Amount:
 If this is NOT a Fixed Price quote, this field will default from the detail lines associated with the Option(s) you selected in the previous screen. It is display only; any changes to the amount should be made to new option’s detail lines after running this wizard.

Scope of Services: This field defaults from the Option(s) you selected in the previous screen. It may be adjusted if needed.

Click the Finish button to complete the copy or Cancel to cancel this operation.
