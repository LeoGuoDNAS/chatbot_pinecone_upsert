Project Manager Workbench Cost Sources Tab

Purpose:

The Cost Sources Tab displays the actual and committed costs associated with this Job.

Usage:

See Project Manager Workbench for details on this viewer’s upper tabs.

Important Fields:

Date – Source – Trx Id – Reference – Hours to Date – Quantity to Date – Actual Cost – Committed Cost - Purchase Order

These costs were written to Job History derived from the following sources: Work Order, Purchase Order, Credit Purchase Order, Labor, Payroll Adjustment and Inventory Adjustment Journals.

If a Purchase Order is associated with a Cost Source in Job History it will be displayed in the Purchase Order column. .
