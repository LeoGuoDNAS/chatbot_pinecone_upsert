Notes Button

The appearance of the Notes button is determined by whether or not Notes already exist for this record.

Â 

When you click the Notes button, a list of the existing notes for this record will appear. You may select an existing Note to view, or click the New Record button to add a new note.

If notes do not yet exist for this record, this version of the button will appear. Click this button to create a new note.
