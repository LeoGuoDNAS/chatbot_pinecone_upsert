Copy Color Button

The Copy Color Palette is located on the second row of the top tool bar in the folders viewer. This feature enables your firm to standardize your Data-Basics colors. When you click this button, a screen will appear enabling you to select the User Id whose colors you wish to copy. If the User Id is blank, clicking OK will reset the default color palette.

Only members of user group 'All' may copy a selected user's colors to all users - regular users not belonging to user group All may only set their own colors from another user.
