Registry ADP Export

Purpose:

Provides required default to support Report ADP Export (see ADP Payroll Export ).

Id: sys-adp-export

Important Fields:

Directory/Filename for Saved Files

Setting Key: PathName  Setting Value: pathtofiles
where pathtoreader is replaced by full path to the directory where you wish to save the export files.

Setting Key: FileId   Setting Value: YY
YY should be replaced by a 1-2 character code used to generate the filename. This is also a field on the input screen. If the FileId is set in the Registry, it will automatically default into the report range screen.

Default Reimbursable Code

Setting Key: DefaultReimbCode Setting Value: XX
XX should be replaced by a 1-2 character code appropriate for ADP. All Reimbursables in v7.0-7.3 will be grouped using this code. In V8.0 and above, the ADP reimbursable code may be set in the Other Rate record, overriding this registry.

Default Company Code

Setting Key: CompanyCode Setting Value: 2-3 character code
This is also a field on the input screen.  If the CompanyCode is set in the Registry, it will automatically default into report range screen.

Reclassify Hours

This modification was developed for specific customers and should NOT be used without first discussing with your SAMPro support rep.  Requires the user to review all Pay Types for Regular and Overtime (hours worked) and change the Hours type for 'Other' to 'Worked' and change the category from 'Other' to 'Regular' or 'Overtime'. After editing these Pay Types, this registry option is added so the ADP export will continue to extract Regular and OT Hours as Hours 3 Amount (prints as Other Hours on the report) but allows the Labor Journal to display the correct Recap totals by Category / Hours Type.

Registry Id: sys-adp-export:

Setting Key: ReclassifyRegOTHours    Setting Value: true  (default is false)

Enabling this registry entry will cause Regular and Overtime hours worked to be reclassified in the .cvs export file as Hours 3 Amount (Other Hours on the report).
