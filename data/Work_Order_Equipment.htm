Work Order Equipment Tab

Purpose:

The Work Order Equipment Tab identifies the work code, work requested and performed, and the equipment that should be serviced or maintained, etc. A virtually unlimited number of Equipment lines may be included on the Work Order Equipment tab.

Example:

Fix a leaking boiler, perform a PM on an air conditioner, etc.

Usage:

Required for Dispatching. Each Work Order can include one or more equipment tab lines.

Important Fields:

Equipment Id:
Enter a valid Client Site Equipment Id or double-click to select from a list. This is a required field.

When you add a new piece of Equipment to the work order, the system will check to see if it includes a Log Form in the Equipment record. If so, the appropriate entries will be made to the Work Order Metrics tab from the Log Form record.

Work Code Id:
This field defaults from the Work Code specified in New Call or Create PM WO but may be changed by the user if needed. Enter a valid Work Code or double-click to select from a list of Work Codes. This field is optional but highly recommended since the Work Code is displayed on the Dispatch Board.

If PCR functionality is enabled, the system will validate the Work Code is specified in the PCR Table (in the Client Site / Client Site Equipment) when a Work Code is entered or changed. The PCR Table in the Client Site Equipment record overrides the PCR Table in the Client Site record.

Cause Code Id:
This field may be used to specify the cause of the problem defined by the Work Code. If PCR functionality is enabled, the system will validate that the Work Code / Cause Code pair is specified in the PCR Table when a Cause Code is entered or changed. The PCR Table in the Client Site Equipment record overrides the PCR Table in the Client Site record.

Resolution Code Id:
Enter a valid Resolution Code Id identifying how this problem was resolved. You may double-click on this field to select from a list of resolution codes. If PCR functionality is enabled, the system will validate that the Cause Code/Resolution Code pair is specified in the PCR Table when a Resolution Code is entered or changed. The PCR Table in the Client Site Equipment record overrides the PCR Table in the Client Site record.

Estimated Hours
The estimated hours to complete this work. This field defaults from the Work Code Id but may be changed by the user if needed. If adding multiple pieces of equipment to a Work Order, be sure to also update the Technicial tab hours that will appear on the Dispatch Board.

Service Parts Id:
The parts list for this task. Double-click on this field to select from a list.

Service Tasks Id:
The service list for this task. Double-click on this field to select from a list.

Wrrnty Strt: WrrntyEnd:
If this equipment is covered under warranty, the start and end dates will default from the Site Equipment record.

Under Contract
This display-only field defaults from the Site Equipment's Under Contract box.

Work Requested:
Describes the work the customer requested for this task; it defaults from the Work Code Extended Description, if applicable. This field is optional but recommended. This field may contain more information than initially appears: position the caret in the field and use the right cursor arrow to scroll and display additional text.

In the case of PM Work Orders, this field may also list the service tasks associated with this equipment; this information is generated when the Create PM Work Order function is run. Note that this default behavior may be turned off by the Registry Entry for Create PM WO .

Work Performed:
The technician's description of the work actually performed. This field is optional but recommended. This field may contain more information than initially appears: position the caret in the field and use the right cursor arrow to scroll and display additional text.

Completed
The date the work on this line was completed. Press Spacebar to insert today's date, or enter the date in mmddyyyy format.

Standard Filters: Belts:
Pleated Filters: Other:
These fields default from the client site equipment record, and specify which part types should be included from the Service Parts List. If corresponding fields are checked, items on the Service Parts List will be included on the parts list recorded in the Description of Work Performed field of PM Work Orders generated for Subcontractors.
