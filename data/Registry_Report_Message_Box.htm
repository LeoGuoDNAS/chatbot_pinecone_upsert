Registry Report Message Box (sys-messagebox)

Purpose:

This registry entry is implemented to allow the report message box to automatically close after running reports.

This will be implemented such that the prompts when running reports will automatically close after a set amount of time, and add a  "Confirm with Timer" prompt that will automatically confirm this prompt after the set amount of time.

The amount of time will be set via the registry defined below. Users can click “Yes” on the message box to close it before it automatically closes. 

Note: This registry is vital for efficiency.

Important Fields:

Id: sys-messagebox

Setting Key: ReportPrintConfirmationTimeout Setting Value: 30 Seconds (0 default)
