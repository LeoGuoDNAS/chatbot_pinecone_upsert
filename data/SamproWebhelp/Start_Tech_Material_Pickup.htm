Start Tech Material Pickup

Purpose:

This starter function is a part of Work Order Material Requisitioning. It enables the transfer of inventory from a STAGING Location to a TRUCK Location (if the Work Order Type is NOT set to Requisition).

If the Work Order Type is set to Requisition, running this function transfers the inventory directly to the Work Order/Job.  The Work Order Type "Requisition" should be used only for the purpose of requisitioning materials for installation / construction jobs and should NOT be used for service or PM WO's.

Usage:

Typically, this function would be run after the Start Release Work Order function sets the Work Order Escalation Status from Waiting Parts to Waiting Service (or Unacknowledged). An Inventory Adjustment Journal entry will be generated when this function is run.

Registry:

The system will pull the default Transfer From (staging) location from the MaterialRequisitionToLocation=staginglocationid option of the Shop Order Registry entry. If Branch processing is enabled, it defaults from the MaterialRequisitionToLocation=staginglocation option of the  sys-viewer-vshpordr-BranchId registry entry.

Important Fields:

Work Order Id:
Enter the Id of the Work Order whose materials should be transferred from the staging area to the Technician’s truck (or double-click to select from a list of Work Orders).

Work Order Type
This display-only field shows the Work Order Type of the selected Work Order.

If the Work Order Type NOT set to Requisition, running this function will transfer the inventory to the selected Transfer To location (typically, the Technician's truck).

If the Work Order Type is set to Requisition, running this function transfers the inventory directly to the  Work Order/Job.  The Work Order Type "Requisition" should be used only for the purpose of requisitioning materials for installation/construction jobs and should NOT be used for service or PM WO's.

Work Order Name: Site Id: Site Name: Location Id:
These display-only fields default from the Work Order record you have selected.

Transfer From:
This display-only field specifies the Staging Location inventory is being pulled from. It defaults from the MaterialRequisitionToLocation=staginglocation option of the sys-viewer-vshpordr (or sys-viewer-vshpordr-BranchId if branches are enabled) registry entry.

Technician Id
This field provides a way of specifying the 'To' Location for non-Requisition type work orders; when the Technician is selected, the Transfer To: location is set to the Tech's truck location.

Location Id:
When the Work Order is first entered, this display-only field defaults from the Work Order header. If the Technician Id is changed, it displays the Tech's truck location.

Transfer To:
If the Work Order Type is NOT set to Requisition, this field should specify the Location Id assigned to the Technician’s Truck. Typically, this would be the same Location Id that was displayed when you selected the Work Order.

If the Work Order Type is set to Requisition, it will NOT be used since the transfer will be made directly to the Work Order / Job.

When all fields are entered correctly, click the Go button to generate the inventory adjustment transfer.
