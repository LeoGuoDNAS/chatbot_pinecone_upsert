Registry Report Function Id (sys-report-ReportFunctionId)

Fillable Forms

Version 9.1.611.051 adds registry-enabled functionality to skip the auto display of the PDF version of fillable forms such as Lien Waiver Report Fillable (AR) report.  If this registry is not in place, clicking the Fillable Form button to generate the formatted, PDF version of the report will open the PDF. If the following registry is in place, the PDF will be created but not opened.

Registry Id: sys-report-reportfunctionid

Setting Key: showdocumentfillableform   Setting Value: false (default true)
