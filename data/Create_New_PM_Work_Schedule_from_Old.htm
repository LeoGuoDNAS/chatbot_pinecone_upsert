Create New PM Work Schedule from Old

Purpose:

Use the existing information in a Client Site Equipment PM Schedule to add new entries to the schedule for the following year or years. This function only clones the PM Schedule on the Client Site Equipment record. To actually create the work orders, you must run the Start Create PM WorkOrder function.

Usage:

This function should NOT be used in conjunction with Master Contract; the Master Contract viewer and / or Renew Master Contract should be used instead.

Master Contract Equipment Table Note: With the UseMasterContractEquipmentTable global registry set to true, lines from the Old schedule are ignored when they have references to Master Contracts. Also note that lines will NOT be copied if the Client Site Equipment parent record references a Master Contract even if the lines in the old schedule do not.

Optional for updating PM schedules. New schedules will only be generated for active equipment (e.g., Equipment whose Inactive field is not checked). In addition, only equipment belonging to Client Sites with a status of Active will be affected by this function.  Alternately, Start Renew Service Contract may be used to update both Client Site Equipment PM Inspections and Client Site PM Billings.

Important Fields:

Years to Add
The number of years (1-9) to add to each existing PM to create the new scheduled entries. For example, if you added one year to work orders scheduled in 2007 the new schedule would be for 2008.

Client Id:
To schedule PM's for a specific Client, enter the appropriate Client Id. To enter a range of Clients, enter the first and last Id's of the range separated by an underscore (ACME_ZENITH). Several Id's can be specified by separating their Id's with a comma (ACME,MESA,ZENITH).

Client Site Id:
To schedule PM's for a specific site, enter the appropriate Client Site Id. To enter a range of Client Sites, enter the first and last Id's of the range separated by an underscore (ACME_ZENITH). Several Id's can be specified by separating their Id's with a comma (ACME,MESA,ZENITH).

Equipment Id:
To schedule PM's for all sites having a piece of equipment with a specific Id, enter the appropriate Equipment Id. To specify only this Equipment Id at a particular Site, enter the Client Site Id in the above range, and the Equipment Id using the following format: &,equipid where equipid is the Equipment Id.

Scheduled Inspection Date:
This field is required: you cannot run this function wide open. This range controls which pm schedules will be copied by selecting a date range. To copy only PM's for a specific inspection date, enter the date in yyyy-mm-dd format. To copy inspection dates for a range of dates, enter the first and last dates of the range separated by an underscore (2007-01-01_2007-12-31). Several inspection dates can be posted by separating them with a comma (2007-01-01,2007-01-03).

Examples:
Suppose that you had scheduled weekly inspections in 2007 for all active equipment at Client Site Id: 1044. To create new PM's for the first quarter of 2008 for this site, you would enter a 1 in the Years to Add field, 1044 in the Client Site Id field, and 2007-01-01_2007-03-31 in the Scheduled Inspection Date field.

Assume that Client Site 1099 has three pieces of equipment, one of which (Equipment Id 1099A) is under a 2 year monthly inspection schedule for 1997 and 1998. At the end of 1998, the customer renews the contract. To create the new pm schedule from the old one, you would enter a 2 in the Years to Add field, 1099 in the Client Site Id field, &,1099A in the Client Site Equipment field and &,1997-01-01_1998-12-31 in the Scheduled Inspection Date field. When you clicked the Go button, the system will display a checklist showing 12 inspections for 1999 and 12 for 2000.

Ok to Renew?
To limit the PM’s created to those with the Ok to Renew? flag set in the PM Schedule, enter a Y in this range.

Important Buttons:

Click to add the selected PM's to the schedule. When you do so, the system will display a list of proposed PM's, based on the parameters that you entered. At this point, you can select the ones that you wish to add to the schedule. See Create New PM Schedule from Old Checklist for details.
