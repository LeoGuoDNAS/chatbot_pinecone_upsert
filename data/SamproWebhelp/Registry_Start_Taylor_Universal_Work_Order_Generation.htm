Registry Start Taylor Universal Work Order Generation

Purpose:

Provide required values for Taylor Universal Registry Entries. This registry is also required to display the UWO Tab on the Work Order record.

Related Topics:

In addition to this registry, certain database fields must be set up correctly for this functionality. See Taylor Universal Work Order Setup for details.

See Start Taylor Universal Work Order Generation for information on running the generate function itself.

See Taylor Universal Work Order Checklist for a handy checklist for users to consult when running this function.

Important Fields:

Registry Id: sys-Taylor-UWO

 

Enable UWO Functionality

Setting Key: Enabled    Setting Value: Yes

Taylor Distributor and Branch

Setting Key: DistributorNo Setting Value: Value supplied to distributor by Taylor

Setting Key: DistributorBranch Setting Value: Value supplied to distributor by Taylor

 

Minimum Line Count:

Setting Key: MinimumLineCount Setting Value: NumberOfLines

Controls the minimum number of lines to include on the Work Order List screen. This defaults to 50 if this registry is omitted.

UWO Default File Path/Filename

Setting Key: UWOFilePath Setting Value: default file path for the generated UWO file

(e.g., N:\sampro\export\UWO.txt)

Specify Vendor Id for Taylor

As part of the UWO, Taylor's part number for a part must be reported. Previously, this could only be obtained from the Inventory Item Vendor table using the Inventory Item's default Vendor. This release provides a registry option to identify the Sampro Vendor Id for Taylor, so that this does not have to be set as the Default Vendor for each Taylor part.

Setting Key: VendorIdForTaylor Setting Value: taylorvendorid

The Setting Value should specify the actual Vendor Id for Taylor in your system.

Cross Reference File Path

Setting Key: CSVCrossReferenceFilePath  Setting Value: full path to directory

The folder identified by the CSVCrossReferenceFilePath points to a folder that should contain several files which together amount to a set of validation rules. These files are supplied by Taylor Corporate and are subject to change. These files should be uploaded frequently.
