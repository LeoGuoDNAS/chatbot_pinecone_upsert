Registry Start Import Stock Levels (sys-start-simprtlvls)

Purpose:

This function is called when the Import Stock Levels is clicked in the Location viewer.

Important Fields

Id: sys-start-simprtlvls

See the Standard Registry section in Registry Viewer for additional Registry Id formats to allow entries for specific Users, Branches, Security, or User Groups to override the main entry.

Limit to Items flagged as Stocked

By default the Picking Sequence, Reorder Level and Order Up To fields for all selected items are imported, regardless of whether or not the item is flagged as Stocked. This enhancement provides a registry entry that will filter out all items not flagged as 'Stocked'.

Setting Key: LimitToStockItems Setting Value: true (default false)
