Job Cost Code Level 2 Viewer

Purpose:

Create and maintain the Cost Code Level 2 records for job cost reporting purposes.

Usage:

Optional. Cost Code Level 2 records are not job-specific, but may be referenced in the Job Cost Code record for reporting purposes. The Cost Code Group and Level2 records enable the user to create levels of costing detail between the Job and Billing Item levels.

Example:

Cost Code Levels 1 and 2 may be used in a hierarchical or non-hierarchical fashion. For example, Cost Code Level 1 could be set up with records for Survey, Engineering, Architectural, and Electrical costs. Cost Code Level 2 could be used to break out expenses by Installations, Services, and so on.

Important Fields:

Note: this Help Topic lists the standard configuration of all fields that may be included in this record. If you are using a customized system, the fields displayed on your screen may not exactly match the order or status (required, optional, default, display-only, hidden, etc.) of the fields described in this Topic.

Cost Code Level2 Id: (15 character, only . and - special characters)
Uniquely identifies this Cost Code Level2 record.

Cost Code Level2 Name: (40 character)
Briefly describes this Cost Code Level2 record. Recommended but not required.

Report Sequence: (numeric format 0000)
This optional field enables you to control the listing order of job costing reports.

Cost Code Level2 User1: Cost Code Level2 User2: (30 character x 2)
These optional fields may be used to record additional information required by your company.

Security Id:
If multiple location security is implemented on your centralized database system, this informational field displays the Security Id of the local user that created this record. If multiple location security is not implemented on your system, or if this record was created by a global user, this field will be blank.
