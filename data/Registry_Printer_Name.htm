Registry Max Lines per Page by Printer (sys-printer-printername)

Purposes:

This registry entry is used to set the maximum number of lines per page in landscape and portrait modes for a printer (typically, this is set globally for a printer but the branch and user Id's may also be specified if needed).

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Setting Value Notes:
The Setting Value is case-sensitive: be sure that you enter the value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.
The equal sign = is a reserved character in the Setting Value. Â It should NEVER be used in the Setting Value field unless you are specifically instructed to do so for particular registry setting.

Important Fields:

Id:

sys-printer-printername-branchid-userid

sys-printer-printername- -userid

sys-printer-printername-branchid

sys-printer-printername-securityid

sys-printer-printername

where printername is the name of the printer exactly as it appears in the Name field of the Windows Printer dialog box. Please note that this field is upper/lower case sensitive. If this entry is specific to a user, the branch must be specified if a user belongs to a branch. If branches are not implemented, two dashes (not separated by a space) must precede the user id.

Specify Lines Per Page

Setting Key: lines-landscape Setting Value: #landscape

Setting Key: lines-portrait Setting Value: #portrait

where #landscape and #portrait are replaced by the maximum number of lines per page that may be printed in landscape and portrait mode respectively.

Example

To set the maximum number of lines to 50 landscape and 64 portrait for a printer named AR Dot Matrix, the Id key field would be entered as sys-printer-AR Dot Matrix and the options are set as follows:

Setting Key: lines-landscape Setting Value: 50

Setting Key: lines-portrait Setting Value: 64.
