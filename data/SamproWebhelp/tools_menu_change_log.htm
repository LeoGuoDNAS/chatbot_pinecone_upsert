Tools Menu Change Log – V6.1 and up 

Important Notice: this help file is intended for use by Authorized Data-Basics Support Personnel only. Incorrect implementation of the information discussed here can create problems with software functionality and/or data.

When you viewing a record or journal entry, the Tools menu includes the ‘Change Log’ option. When you select this option, a list of the users who have modified the parent record will be displayed, along with the date and time of the modification (e.g., with the ‘Save’ button was pushed). Changes to child tables (where the parent record was not impacted) are not recorded in the change log. Information regarding which fields have been changed is not recorded.

In version 7.2 and up, an entry to the Record Change Log table is now recorded when a record is added.

Changes made to a record by running a system feature are not fully supported as yet: omissions should be reported to Jean Knox or Jeff Carr.
