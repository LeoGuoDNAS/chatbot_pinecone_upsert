Registry Deny Deletion of Lines (sys-viewer-afunction)

Purpose:

This registry entry provides a generalized mechanism for denying line deletes in SAMPRO viewers with named child tabs.

Usage:

Testing Your Registry Entries
The system does not verify the validity of registry entries. Be sure to test your registry entries after completing them to verify that they are functioning correctly.

Setting Values are Uppercase-Lowercase Sensitive:
The Setting Value is case-sensitive: be sure that you enter the value exactly as specified in the documentation. If the documentation specifies a value of true, do NOT enter True or TRUE.

Important Fields:

Id: sys-viewer-afunction

Setting Key: DenyLineDelete Setting Value: list of tabs

List of tabs should be replaced by a comma-delimited list of the tab titles of tabs for which the delete line function should be denied.

A sample registry entry is shown below.

Id: sys-viewer-vwrkordr
Setting Key: DenyLineDelete Setting Value: Technicians,Equipment

The Setting Value string is a comma-delimited list of child tabs for which the line delete action should be denied.  A shorthand entry for denying all lines deletes in a viewer can be made by setting the value string to 'All'.

NOTE: this option does not override any hard-coded rules already implemented in SAMPro.
