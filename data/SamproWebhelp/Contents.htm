What Version of SAMPro Am I Running?

If you select the Help menu and choose the About this Product option, the system will display an information box listing the SAMPro version number, date, and database revision level. If you choose the About this Product option from a User Defined Report window, the Report Id and Revision Level of the report will also be displayed. Note that system (Smalltalk) reports do not contain this information.

WebHelp

Data-Basics offers support for context-specific WebHelp. WebHelp is activated via a registry option to your sys-ffv-global-settings registry:Global Settings for WebHelp . WebHelp will be launched with the user's default web browser.

Updates: If WebHelp is updated, the first user into the system after the update will be asked to confirm the update: they should always answer ‘Yes’ to this confirmation.

User Requirements:

To see the WebHelp correctly in Internet Explorer, the "Allow active contest to run in files on My Computer" needs to be checked (Internet Options => Advanced => Security) . Otherwise, you will have click on the browser to allow active content every time you access Help.
