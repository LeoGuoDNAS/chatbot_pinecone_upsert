Registry Report Change Order Request

The revised version of Report Change Order Request (RCHNGEORDRRQST.rpt) dated 2021-03-12 supports several bitmap overrides based on fields in the Job or Site. The bitmap registry option will be used to specify the bitmap if an override is not found.

Id: sys-report-rchngeordrrqst

Override Based on the COR's Job / Site

Setting Key: BitmapOverrideBasedOn    Setting Value: branch

Setting Key: Branch~RecordId  Setting Value: sharepath\bitmap.bmp

The first option turns on the override behavior; the Job's Branch will be used to specify the bitmap. The second option specifies the bitmap that will be used with a particular branch.  If the Job record's Branch is not completed, the Site's Branch will be used to determine the override.

Override Based on the COR's Job

Setting Key: BitmapOverrideBasedOn    Setting Value: job

Setting Key: Job~JobId  Setting Value: sharepath\bitmap.bmp

This registry would only be used if certain Jobs needed a specify bitmap.

Overrides Based on the COR's Client Site

Setting Key: BitmapOverrideBasedOn    Setting Value: FieldInClientSite

This option turns on the override behavior, and defines which record in the Client Site will be used to define the bitmap override. Valid values are:

accountmanager
servicemanager
dispatcher

Setting Key: FieldInClientSite~RecordId  Setting Value: sharepath\bitmap.bmp

This entry defines matches the override with the bitmap.  One entry needs to be created for each override bitmap. The Setting Key defines the field and individual record Id; the Setting Value defines the bitmap.  

Example:  Use the Client Site's Account Manager to Specify the Override

In this example, assume that Jean,Tim, Joe and Sarah are account managers. Jean and Tim need special overrides; Joe and Sarah do not.

Id: sys-report-rchngeordrrqst

Setting Key: BitmapOverrideBasedOn    Setting Value: accountmanager

Turns on the override.

Setting Key: AccountManager~Tim  Setting Value: sharepath\Tim_invoice.bmp

Setting Key: AccountManager~Jean  Setting Value: sharepath\Jean_invoice.bmp

Specifies the override for CORs for Client Sites with Tim or Jean as Account Managers.

Setting Key: bitmap    Setting Value: sharepath/Company_invoice.bmp

Provides the default bitmap for Account Managers Joe and Sarah, and for COR's generated for Client Sites without an Account Manager specified.
