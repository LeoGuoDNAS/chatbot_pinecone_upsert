Start Compute Technician Evaluation

Purpose:

This function calculates a summary score for employee and/or vendor technicians based on the information in the Technician Evaluation tab of the Work Order. This summary score will be displayed on the Evaluation tab of the Technician viewer.

Usage:

In order for this function to accumulate data correctly, the Evaluation Id must be set in the Main and PM Schedule tabs of the Client Site Equipment record so that this field will default to all work orders.

Important Fields: Initial Screen

Start Date: End Date:
The time period to be evaluated. Work Orders that were created within this date range will be evaluated.

Evaluate Employee? Evaluate Vendor?
One or both of these boxes must be checked (by double-clicking) to run this function. If the Evaluate Employee box is checked, primary work order technicians that link to an employee record will be evaluated. If the Evaluate Vendor box is checked, primary work order technicians that link to a vendor record will be evaluated.

Technician Id
To limit this evaluation to a specific Technician, enter an Id.

Click the Go button when all fields are completed.

Important Fields: Checklist Screen

Do
This field enables you to select which Technicians will have their evaluations posted. By default, all Technicians are selected.

Double-click on the Do field to select or deselect an item (alternately, press the spacebar when the caret appears on this field); a check appears in the Do field when that technician is selected. Refer to the Important Buttons section of this Help topic for details on selecting or deselecting all items.

Tech Id: Eval:
These fields are displayed for informational purposes so that you can verify the technicians for whom the evaluation will be completed. The Evaluation field is computed by the system based on the number of points that were earned divided by the number of points available during the time period you specified in the previous screen.

Comments
You may add your comments on this evaluation if desired.

Important Buttons:

Click the Check button to check (select) every item on the list.



Click to de-select every item on the list. Click to complete the evaluation of the selected technicians.
