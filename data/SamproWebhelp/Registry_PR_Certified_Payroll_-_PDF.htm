Registry PR Certified Payroll - PDF

The following steps are required to implement this functionality in SAMPro 9.1.611.055 and above. See Report PR Certified Payroll - PDF for details on running the report and generating the PDF.

1. The xmltopdf.jar file must support Java 8 (remember the file must be rename XmlToPdf.jar):

Java 8-11: TLS 1.2 and Uses Javamail 1.6.2 (XmlToPdf[2019-04-05 Java 8-11].jar) or

Java 8-11: TLS 1.0:  XmlToPdf[2019-01-16] Java8-11].jar

2. This functionality is dependent on database functions; contact your support person to deploy script attached to Devops t4427.  

3. Install Report PR Certified Payroll PDF (RPRCRPRPDF.rpt dated 2021-02-08 or later and assign to appropriate User Groups.

4. Copy the PDF template files, drop the wh347section1.pdf and wh347section2.pdf files in an appropriate templates directory on your server.

5. Registry Setting

Id: sys-report-rprcrprpdf

Setting Key: fillableform  Setting Value: true

Setting Key: sectionfillableform  Setting Value: true

Setting Key: SaveFillableBySection  Setting Value: false

Note that this setting allows the two-part fillable WH 347 to be saved as a single document.

Setting Key: sectionfillableformwh347p1  Setting Value: fullpath to template\wh347section1.pdf</Line>

Setting Key: sectionfillableformwh347p2  Setting Value: fullpath to template\wh347section2.pdf</Line>

These options specify the templates for page 1 and 2 of the report.

Setting Key: sectionfillableformflatten  Setting Value: false

Setting Key: autosavebysection   Setting Value: false

Note that this setting allows the two-part report to be saved as a single document.

Setting Key: autosavedirectory  Setting Value: fullpath to save directory

Setting Key: autosavefilename   Setting Value: wh347 [^date^^time^]

Setting Key: signatory  Setting Value: Name of Signatory to appear on report

Setting Key: title     Setting Value:  Title of Signatory to appear on report

Setting Key: contractor   Setting Value: Name of contracting company (optional)
