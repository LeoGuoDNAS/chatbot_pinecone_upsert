Technican Dispatch Board Tab

Purpose:

The Dispatch Boards tab on the Tech record provides right-click add or delete of Technician from the from the View Boards and Technicians viewer. Added Techs will be added to the end of the Tech list in View Boards and Technicians. The record change log of View Boards and Technicians is updated when a Tech is added or deleted.

Important Fields:

Serves
If the contents of this field is set to Yes, the Technician is affiliated with this Dispatch Board. If the Serves field is blank, the Technician is not associated with this Dispatch Board.

To set the Serves field to Yes, right-click on the line you wish to change. Right-clicking will also toggle a Yes to a blank if you wish to remove the Dispatch Board affiliation.

Id - Name
The Dispatch Board Id and Name. Double-click on a line to access the Boards and Technicians viewer.
