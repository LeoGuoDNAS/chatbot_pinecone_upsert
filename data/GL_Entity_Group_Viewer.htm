GL Entity Group Viewer

Purpose:
 May be used to group GL Entities for reporting purposes.

Example:
 Group service profit centers (GL Entities), manufacturing profit centers, etc.

Usage:
 Optional for financial reporting.

The entities that belong to this group may be defined in one of two ways:

By defining them in the GL Entity Id Like String

Or by entering them in the GL Entity table in the lower half of the screen.

Important Fields: Upper Section

GL Entity Group Id: (10 character, only . and - special characters)
A unique Id for this GL Entity Group. This field is required.

GL Entity Group Description: (40 character)
A brief description of this GL entity (optional).

GL Entity Id Like String:
This field should not be used if Intercompany Security is enabled. Depending upon how your GL Entity Id's are set up, this field may be used to define which GL Entities are included in this group.

Example:

Suppose you have six GL Entities:

S1, S2, S3, P1, P2, P3

If you wanted to include GL Entities S1, S2 and S3 in this group, you would complete this field as follows:  S%

This would tell the system to include all GL Entity Id’s beginning with S in this group. If you later added the GL Entity S4, it would automatically be included in this group.

If your GL Entity Id naming structure will not work with this field, simply add all appropriate GL Entity Id's to the GL Entity table in the lower section of the screen.

Company Id:
Although a specific Company Id may only be associated with one GL Entity Group, other GL Entity Groups may be created for GL Reporting purposes without the Company Id completed.  The GL Entities tab should be used to specify the GL Entities.

Security Id:
If multiple location security is implemented on your centralized database system, this informational field displays the Security Id of the local user that created this record. If multiple location security is not implemented on your system, or if this record was created by a global user, this field will be blank.

Important Fields: Lower Section

GL Entity Id:
Enter a GL Entity Id that belongs to this group or double-click to select from a list. If you enter GL Entities in this table, you should not complete the GL Entity Id Like String field in the upper section.
