Write-off Using Discount Method

Assume that during application of cash receipts.. . I need to write off $700 from 2 invoices. Enter check amount as “Total Received”. Enter the total amount to be written off into “Total Discount”. By invoice, enter amount to be written off using ‘discount taken’. Before adding trx, click on the GL TAB (next to Main tab)….

 

On the GL TAB….. change the GL Account number for DISCOUNTS to your WRITE OFF GL Acct #:

 

The following type of G/L Journal Entry will be created when user saves Cash Receipt entry:

q   Debit to cash or bank sweep account (1000.3) for the amount of the customers check

q   Debit the “write-off” account (6040) for the amounts being written off (total discounts taken)

q   Credit A/R (1100) for the original invoice amounts

 

NOTE: When changing the GL TAB…… these are ‘sticky’ fields by user. The next time this Cash Receipt program is used by the user, the user must change back the GL account number to the DEFAULT Discounts allowed account, otherwise the system will continue to use the stored GL account last entered. (e.g. Sales tax, employee loans, bad debt, etc).
