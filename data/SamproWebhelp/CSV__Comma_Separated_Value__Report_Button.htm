CSV (Comma Separated Value) Report Button

The .CSV button appears on the list view of all record viewers. It is designed to create a comma separated value text file with the contents of the list view itself. CSV files may be imported into directly into Excel spreadsheets. This button is only enabled if the Userâ€™s Group and Functions record allows Export of this record.

When you click this button, the system allows the user to "point" to the folder in which you want to save the file. This enables network users to save the file to their local hard drive. The file name can also be changed.
