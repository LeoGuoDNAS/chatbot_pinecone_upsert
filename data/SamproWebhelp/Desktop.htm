Desktop

Purpose:

The Desktop allows you instant access to those functions and reports that you use most frequently.

Usage:

If a custom desktop has been defined for your User account, your own user desktop will be displayed. If a desktop has not yet been defined, the Folders viewer will appear instead.

Important Fields:

The desktop window contains a customized list of viewers, reports, and starter functions. This list may be organized by user-defined topics. To display a function on this list, simply double-click on it. Alternately, click on the function once to focus on it and then click on the Go button.
