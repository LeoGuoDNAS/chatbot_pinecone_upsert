Start Export Report

Purpose:

Since user Defined reports are stored in your database, this new support function enables you to export one or many of your user-defined reports from your database to a directory on your disk. Reports should be exported to a folder for archival purposes before importing new ones. Your support representative will inform you if there are other times when it is necessary to a report (typically, this would be done for support reasons).

Usage:

To export reports, complete the following steps.

1. To export your user-defined reports, create a directory you can export them to when you are running the Data-Basics software. For example, if you are running the software from c:, you might want to copy the report files c:. If this directory contains earlier copies of reports, we suggest that you move these files to a backup location (c:).

2. Access the Report Export function. This function may be accessed from your list of Starter functions or from the Folders viewer (the Reporting tab of the Utilities topic). Fill in the Important fields as discussed below.

3. When you click the Go button, the system will display a checklist containing all of the reports that you specified for export.

4. When you click the Go button on the checklist, the system will begin the export. Once the export is complete, the report will be saved to the directory that you specified.

Important Fields:

Reports Directory:
Enter the path to the directory containing the reports (for example, c:).

Revision Level:
The version number of the report to be exported. This field acts as an optional filter to the export function, so that only reports corresponding to this version will be exported.

Report Id:
This field acts as an optional filter to limit the reports to be exported. Enter the Id(s) of the report(s) that you wish to export. You may double-click on this field to select from a list (use a comma to separate multiple names).

Report Name:
This field acts as an optional filter to limit the reports to be exported if you do not care to specify the Report Id's. Enter the Names of the reports that you wish to export (use a comma to separate multiple names).
