Report PR Canadian T4s (Standard)

This report generates the Canadian T4 forms; it does NOT require a printer with Native PCL 5.

Usage:

The registry options discussed in Registry T4 Print (sys-printer-T4) and Registry Report PR Canadian T4 (Standard) must be in place for this report to print properly. The Registry Report PR Canadian T4 Summary (Standard) adjusts the T4 Summary.

Important: Deduction Record Setup:
In order for Union Dues and Charitable Deductions to correctly print on the T4, the user must populate the field ‘Deduction User 1’ with Dues or Charitable, respectively, and ensure that the Deduction Type is Other Pre-Tax.

Earnings and Deduction Records – Other Information
Canadian T4's use the W2 Category fields in the Deductions & Earnings records to populate the Other Information boxes on the T4. The W2 category must be filled in as OI:(Code # for T4). The Employee User 7 field is used to print the registration number in box 50 on the T4.

Box 14: Supports the processing of exempt earnings for the calculation of Employment Income.

Box 20: Enter RPP Contribution amounts using an Earnings Id with the W2 Category set to 20.

Box 24: will be blank if it is the same as Box 14.

Box 26: will be forced to blank.

Box 52: Enter Pension Adjustments using Earning Id's with the W2 Category set to 52.

Canadian PR T4 Standard and Summary - Covid Reporting for 2020

This release supports Covid 19 reporting for 2020. This is implemented with special, year-specific earnings processing. The Other Information boxes will be used to print totals / codes as follows based on the Check's Effective Date:

Code - Check Date Range

57 -  2020-03-15 through 2020-05-09
58 - 2020-05-10 through 2020-07-04
59 - 2020-07-05 through 2020-08-29
60 - 2020-08-30 through 2020-09-26

Special Processing for Partial Year Provincial Reporting When an Employee Has Moved

If an employee moves from one province to another during a calendar year, special processing must be followed in order to produce the correct printed T4's for that employee.

1. The registry sys-report-rt4std must include the following setting to allow the Start and End Dates to be edited when the T4 is run:   Setting Key: PartialYearReporting  Setting Value: true

2. Determine when the Provincial Tax in the Employee's Tax Tab was changed from Province 1 to Province 2. You will need to determine the last Payroll Period the employee's check was printed with their Province set to Province 1.

3. Run the Employee's T4 with the End Date set to the last Check Date earned in Province 1, and the Province field set to Province 1.

4. Run the Employee's T4 with the Begin Date set to the day after the last Check Date earned in Province 1, and the Province field set to Province 2. The End Date should be set to year-end.

Important Fields:

Copy to Print:
Select either Employer or Employee version:

Employer version prints two employees per page; generates the T4 Summary page

Employee version prints same employee twice on one page; does not generate the T4 Summary page

Province:
Select the Province.

Year:
 Select the Year: the Start Date & End Date range will default to Jan 1 – Dec 31. See above for special processing if an employee has moved from one province to another during the year.

Plain Paper copy?

The first print should be to plain paper to check the numbers. This functionality is not needed in this version of the report since the report will always be displayed.

Enter T4 Summary Remittance:
This amount will balance to the amounts on the T4 Summary. It is not necessary to enter this amount when running Employee version as only the Employer version generates the T4 Summary Page

Employee Id:
Enter Selected Employees (or list of Employees)  or leave Blank for all employees.

Executive Payroll
Enter Y if this is an executive payroll or clear for all payroll types. Note that only Executive Payroll users will be able to print forms for Executive Payroll employees. The Executive P/R status of the Payroll Period used to record entries is not considered.

Running the Report:

Step 1. Run Report PR Canadian T4 (Standard) and print a test alignment of your tax forms for one or two employees before running all copies. You will be prompted to run the T4 Summary (although Report PR Canadian T4 Summary (Standard) may be also be run stand-alone).

Make any adjustments needed to the horizontal or vertical positions via the sys-printer-t4 and/or sys-printer-t4summary registries and recheck for alignment as needed.

Step 2. Once you are happy with your alignment, run the necessary copies of your T4 forms. A message will be displayed while the forms are still printing:

Print T4 Summary Form? Insert Form Now.

If you want to print the T4 Summary, DO NOT RESPOND TO THIS MESSAGE IMMEDIATELY. The user should verify that all T4's have been printed, and then load the T4 Summary form in the printer.

Step 3: After the T4 Summary form has been loaded, the user should then click Yes in the message box to print the Summary.
