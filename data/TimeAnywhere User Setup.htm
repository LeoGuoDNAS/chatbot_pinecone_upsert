TimeAnywhere User Setup

Configure TimeAnywhere User:

  1. Add an Employee record for the TimeAnywhere user. On the Other Tab, the TimeAnywhere Type will need to be setup:

    Individual: Can only enter time for themselves.

    Foreman: Can enter time for themselves or other employees.

  3. Add or edit a User record for the TimeAnywhere user.

     - User ID                  (used for TimeAnywhere login)

     - Link to Employee ID       (created above)

     - Set Password             (used for TimeAnywhere login)

     - Set Access Type = Remote (TimeAnywhere licensing)

    NOTE: Can use existing Full/ReadOnly licenses

  5. Optional Link a Technician record to the Employee record to enable the “My Work Orders” search option. TimeAnywhere users will then be able to enter time in for work orders assigned to them.
