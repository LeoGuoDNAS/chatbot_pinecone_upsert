Remote Access Query: wo_closing_script_list

Purpose:

Defines the contents displayed in the Closing Call Script table in TechAnywhere.

Usage:

The headings of the closing call script table must be defined by the Remote Access Query: wo_closing_script_headings .

Important Fields:

Application: TechAnywhere

Dataset Name: wo_closing_script_list

Query Id: This field is assigned by the system when the record is saved.

Query Format: SQL

DataSet Query:

select

wrkordr_id as Work_Order_Id,

cllscrptlst_qstn as Question,

'' as Response,

cllscrptlst_ln as Ln

from wrkordr, wrkordrstts, tchncn, cllscrpt, cllscrptlst

where tchncn_id='^tchncn_id^'

and wrkordrstts_stts_ctgry='open'

and wrkordr.wrkordr_rn=wrkordrstts.wrkordr_rn

and wrkordrstts.tchncn_rn=tchncn.tchncn_rn

and wrkordr.clsng_cllscrpt_cllscrpt_rn=cllscrpt.cllscrpt_rn

and cllscrpt.cllscrpt_rn=cllscrptlst.cllscrpt_rn

and wrkordr.wrkordr_rn<>0

and cllscrptlst_qstn<>''

order by wrkordr_id, cllscrptlst_ln
