Magnify Button

Many tabular views enable you to switch from the Line View (often requiring scrolling to view the right-hand fields) to a Magnify (Detail) View of one line. Single (Detail) View is a "stacked" view of that line in which all data-entry fields are displayed simultaneously so that scrolling from side to side is not required.

To display the Single (Detail) view of a line, click the Magnify button in the first column of the line that you wish to display. When you do so, the Single (Detail View) for that line will appear. See Single (Detail) View Buttons for details on operating this mode.
