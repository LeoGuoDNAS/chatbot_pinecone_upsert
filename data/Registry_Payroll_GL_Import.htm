Registry Payroll GL Import

The Start Payroll GL Import function allows for the import of a PR GL file from a payroll service; currently, Wells Fargo is supported. A GL Journal entry will be created when this function is successfully run.Note that the v.rgstry must have PRInterface enabled in order to run this function.

The following registry is required to import payroll data from Wells Fargo.

Registry Id: sys-start-sprglimprt

Setting Key:  Format    Setting Value: WellsFargo

The Wells Fargo import file is comma-delimited. The first row is a header row, and is ignored by the import.

The columns are defined as follows:

Columns 1-2: Wells Fargo info, skipped by the import

Column 3: GL Account Id

Column 4: GL Entity Id

Column 5: Debits

Column 6: Credits

Column 7: Description (skipped by the Import)

Columns 8-10: Skipped by the Import but must be included
