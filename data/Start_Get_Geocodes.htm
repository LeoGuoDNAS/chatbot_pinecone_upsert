Start Get Geocodes for Client Sites

Purpose:

This starter function may be run to automatically GeoCode Client Sites (supports SageQuest / Fleetmatics). It allows the user to filter by:

Site Id
Branch Id
Account Manager Id
City
State
Active?

Usage:

This function will only process records that have a blank Lat/Long value in the Site record.

Client Site Templates are excluded from this function.

The GeoCode search will now send along the city and state portions of the address to assist in the latitude/longitude search.
