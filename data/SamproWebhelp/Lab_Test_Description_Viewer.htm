Lab Test Description Viewer

Purpose:

Used to create and maintain the individual lab tests specified in the Test Method List (Lab Test Profile) and Lot Master Record for Quality Controlled items.

Usage:

Referenced by: Lab Test Profile (Test Methods List), Lot Master (Inventory Items by Test).

Important Fields:

Note: this Help Topic lists the standard configuration of all fields that may be included in this record. If you are using a customized system, the fields displayed on your screen may not exactly match the order or status (required, optional, default, display-only, hidden, etc.) of the fields described in this Topic.

Lab Test Description Id: (40 character)
This required field uniquely identifies the lab test (e.g., % Solids, Chromaticity X, etc.).
